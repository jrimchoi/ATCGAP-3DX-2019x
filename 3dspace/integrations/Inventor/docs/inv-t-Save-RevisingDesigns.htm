Revising Designs

The Revise option is available upon right-click on new and modified designs in the ENOVIA Save dialog box.

Before you begin:

Ensure that the revision sequence is set in the ENOVIA Save Options dialog box.

  1. Right-click the selected designs and click Revise.

  2. A new column, Target Revision is added in the ENOVIA Save dialog box. The target revision specified in the ENOVIA Save Options dialog box is set as the revision.

    You can edit the values of the revision specified under the Target Revision column.

  3. Click Refresh. The rows for which revisions are selected are refreshed.

    Note:


    • You can enter any existing (stored in ENOVIA) or new revision in Target Revision column.
    • It is not mandatory that the subsequent revisions specified are in sequence. For example, the specified revisions can be of the order, A, B, and F.
    • Once a higher revision label is assigned, then you can assign only an existing earlier revision to the designs. For example, if the existing revisions are A, B, and F, then the next available revision is 'G'. The revisions C, D, and E cannot be used. The existing revisions, A,B, and F can be reused.
