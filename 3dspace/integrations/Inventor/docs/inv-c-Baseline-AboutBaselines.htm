About Baselines

Baselines are used to capture a design or design structure which the designer feels can be used as a benchmark before proceeding to make more changes to the design.

Using ENOVIA Collaborative Design for Inventor, baselines can be created to capture the structure of a design active in Inventor that exists in the ENOVIA database. The baselined structures can be opened at any stage of the design in Inventor from ENOVIA Open or ENOVIA Insert dialog box.

When a design that is opened from ENOVIA is baselined, the baseline is created for the as stored structure of the design in ENOVIA database. For example, if you create a baseline for a design that was opened from ENOVIA and modified, then the baseline does not capture the modifications because they do not exist in the ENOVIA database. The baseline is created for the stored design in ENOVIA.
