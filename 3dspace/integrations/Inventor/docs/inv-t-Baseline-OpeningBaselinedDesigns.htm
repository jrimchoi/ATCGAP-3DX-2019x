Opening Baselined Designs

This task shows you how to open a design from ENOVIA in Inventor using an existing baseline.

Before you begin:

Baseline must exist for the design.

  1. Click ENOVIA > Open in Inventor toolbar to open the ENOVIA Open dialog box.

  2. Search for the baselined design in the ENOVIA Open dialog box. See Searching Designs.

    The results are listed in the ENOVIA Open dialog box.

  3. Right-click the name of baselined design and click Baselines > <Name of the Baseline>.



    If multiple baselnes exist for the design, then all the baseline names are listed.

    Baselines are applied only to the node selected during creation and the structure beneath the node does not inherit the baseline.

    The design opens in Inventor retaining the design's characteristics at the time the baseline was created.
