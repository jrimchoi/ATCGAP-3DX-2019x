Using Autoname

The Autoname option is available depending on the Preferences set by the Integration Administrator.

Only designs with status "New" can be named using Autoname.

Before you begin: The Autoname series must be set in the ENOVIA Save options dialog box.
  • In the ENOVIA Save dialog box, right-click designs with status "New" and click Autoname.

    Names are generated automatically according to the selected Autoname series in ENOVIA Save options dialog box and are displayed under the Autoname column in ENOVIA Save dialog box. On saving, the designs are saved with the auto generated names.
