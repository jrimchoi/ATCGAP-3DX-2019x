Connecting to ENOVIA

After you install ENOVIA Collaborative Design for Inventor, the ENOVIA menu is displayed in the Inventor application.

Before you begin:

Inventor User role must be assigned to your Username. For more details, see ENOVIA Collaborative Design for Inventor Installation and Administrator's Guide.

  1. Click ENOVIA > Login in the Inventor tool bar.

    Inventor connects to ENOVIA if an active session exists.

  2. ENOVIA Login dialog box opens.



    Enter your User Name and Password and click Login.

  3. You are logged in to ENOVIA. To check whether your log in is successful, click ENOVIA from Inventor toolbar. All ENOVIA commands except Login are enabled.

    If you are already logged in to ENOVIA Designer Central, click Connect to join an existing session.

    A is displayed upon connecting to ENOVIA.
