Saving Designs in ENOVIA

The Save function lets you save the design information into the ENOVIA vault for sharing across the enterprise. The following sections describes some concepts of Save function and steps to save new designs, modified designs, configurations, and also save designs without using user interface. This feature is used to save new and modified designs to ENOVIA store. The default settings that must be applied and derived outputs that must be generated while saving designs to ENOVIA are controlled by your preferences settings.
