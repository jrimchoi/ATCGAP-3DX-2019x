Attachments

You can attach any external file to a part, assembly, or a drawing in Inventor.

When you checkin or checkout a part, assembly, or drawing which has a linked attachment, then the attachment is also checked in or checked out along with the model.

  • Checking In a Model with an Attachment
  • Checking Out a Model With an Attachment

Checking In a Model with an Attachment

To include an attachment of a model during checkin, you must set the supportattachments property in the settings.ini file of the ENOVIA Collaborative Design for Inventor to true.

The value of this property is false by default. For more details on the settings in settings.ini file, see ENOVIA Collaborative Design for Inventor Installation and Administrator's Guide.

Click ENOVIA > Checkin for a model with an attachment and the Checkin page opens.

The Checkin page displays the model Checkin page and you can select the model for checkin.

Selecting Delete Local Files on Checkin option in the Checkin page, deletes the attachment along with the model after checking in the model with the attachment.

Click Checkin to checkin the model and the linked attachment is silently checked in with the model.

Checking Out a Model With an Attachment

Checking out a model with an attachment includes the attachment for checkout.

The Checkout page does not display the attachment in the structure of the model displayed.

The attachments are checked out with the model to the same directory as the model files.
