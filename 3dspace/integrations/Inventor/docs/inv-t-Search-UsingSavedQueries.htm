Using Saved Queries

This task shows you how to run a saved query from the ENOVIA Open dialog box.

Before you begin:

The searches must be saved previously

  1. Click Saved Queries tab in ENOVIA Open dialog box.

    The saved queries are listed.

  2. Double-click the name of the saved query or select a query and click to execute the query.

    The search results for the query is listed in the ENOVIA Open dialog box.
