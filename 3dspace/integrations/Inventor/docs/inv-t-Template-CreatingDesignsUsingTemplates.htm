Creating Designs Using Templates

This task shows you how to create designs using predefined templates.

Templates are created by the Integration Administrator or an user who has appropriate access privileges. See "Defining Start Design Templates" section in ENOVIA Designer Central Installation and Administrator's Guide for details on creating templates.

Before you begin:

Required templates must exist.

  1. Click ENOVIA > New from Inventor toolbar to open the ENOVIA New dialog box.



    Enter details for the new design including the following:

    Templates. Select a template listed for the Inventor design to be created. The Attribute section lists the attributes of the selected template.

    Specify the Name of the design using one of the following methods:


    • Name. Enter a name for the design.
    • Autoname. Select the Autoname check box and select the naming series from the drop down list. Selecting the Autoname checkbox disables the Name field

  2. Click OK when done.

    The template file is downloaded to the Checkout directory defined in the Preferences and opened in Inventor as the active session.

    The design is not saved to ENOVIA until you save the design using ENOVIA > Save option.

    The new design is created with the selected template and is loaded in Inventor.
