Overview

ENOVIA Collaborative Design for Inventor provides a multi-site Inventor design data management solution for the extended enterprise. It allows designers to access and share each other's designs from within the native Inventor user interface by leveraging the design team collaboration capabilities of ENOVIA Designer Central.

The ENOVIA Collaborative Design for Inventor User's Guide is intended for users who need to become quickly familiar with the xxx product.

Collaborative Design for Inventor in a Nutshell

ENOVIA Collaborative Design for Inventor puts access to ENOVIA capabilities in the Inventor user interface, allowing designers to effortlessly access, manage, share, and store CAD design data without leaving their preferred environment. ENOVIA Collaborative Design for Inventor facilitates process workflow, increases data integrity, and improves configuration management.

From within ENOVIA Collaborative Design for Inventor, users connect to the database using Designer Central engine predefined access setup to search and browse, lock and check-out designs including associated drawings to their local drives. After modification, users again utilize capabilities provided by ENOVIA Designer Central to store changes in the ENOVIA database.

The ENOVIA Collaborative Design for Inventor supports the following languages:

  • English
  • Japanese
  • French
  • German
  • Italian

Before Reading this Guide

Before reading this guide, you should be familiar with basic Version 6 concepts such as document windows, standard and view toolbars.

You should already:

  • Know how to use Inventor
  • Have the ENOVIA Integration Framework Client installed
  • Have the ENOVIA Collaborative Design for Inventor installed
  • Be assigned a username, password, roles and access privileges to the objects you need to work with.
