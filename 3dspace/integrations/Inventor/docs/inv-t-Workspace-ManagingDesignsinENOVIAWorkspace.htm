Managing Designs in ENOVIA Workspace

This task shows you how to manage designs using ENOVIA workspaces.

Before you begin:

Integration Exchange Framework client must be installed.

  1. Click ENOVIA > Manage Workspace > ENOVIA from the Inventor toolbar.

  2. ENOVIA Integration Exchange Framework Client opens.

    The location ENOVIA workspace opens depends on the status of the design in the CAD tool and in ENOVIA. The following table lists the actions of ENOVIA workspace depending on various design status:

    Design Status

    Action

    Active In-session and in ENOVIA

    Opens the ENOVIA folder connected to the active in-session design.

    New design or not connected to ENOVIA folders

    Opens top level of the ENOVIA database locations.

    No active in-session design

    Opens top level of the ENOVIA database locations.

    ENOVIA in-session active design, No Access to Workspace

    Opens top level of the ENOVIA database locations.

    In-session active design, design connected to multiple Workspace folders

    Opens top level of the ENOVIA database locations.

  3. You can use these folders for managing the designs.

    For more details on working with ENOVIA Integration Exchange Framework client, see Working with ENOVIA Integration Exchange Framework Client section in ENOVIA Designer Central User's Guide.
