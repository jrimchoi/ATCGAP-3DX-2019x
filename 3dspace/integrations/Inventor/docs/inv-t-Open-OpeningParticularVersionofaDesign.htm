Opening Particular Version of a Design

This task shows you how to open an earlier or later version of a design from ENOVIA in Inventor.

This feature is used to retrieve an earlier design or a later version of a design if it exists to restart development.

Before you begin:

Multiple versions of the design must exist in ENOVIA

  1. Click ENOVIA > Open or Insert from Inventor tool bar to open the ENOVIA Open dialog box.

  2. Search for the design using Search option in ENOVIA Open dialog box. See Searching Designs.

    The search results are listed in ENOVIA Open dialog box.

  3. Right-click a design and click Open Version > <version number> of the design.

    This option is not displayed if multiple versions of the design does not exist.

    The selected version of the design is opened in Inventor.
