Using Quick Save

This task shows you how to save designs to ENOVIA without opening the ENOVIA Save dialog box. The Global preferences set by the Integration Administrator are applied to the designs on saving.

Before you begin: The Check in preferences must be set by the Integration Administrator.
Related Topics
About Saving in ENOVIA
  • Click ENOVIA > Quick Save from AutoCAD toolbar.

    ENOVIA Save dialog box opens if any obsolete nodes are present in the designs being saved.

    ENOVIA Progress Status shows the progress or status of the data transfer. Upon saving successfully, a success message listing the saved designs opens.

The selected designs are saved with the values set by the Integration Administrator in Check in Preferences.
