Locking and Unlocking Designs

This task shows you how to lock or unlock designs in ENOVIA from AutoCAD. This is used to lock the designs after you have made changes and to prevent other users from modifying the design.

An object that is locked by another user can be unlocked only if the person has unlock access and is assigned the Integration Administrator role.

The status of the native file changes to read-only if the object is unlocked (meaning the user cannot edit the file) and the file status is read-write when it is locked (the user can edit the file).

Before you begin: Ensure that you are in the ENOVIA Open or ENOVIA Insert dialog box.

About Locking and Unlocking

  1. Right-click one or more selected designs in ENOVIA Open dialog box.



  2. Click Lock or Unlock depending on the lock status of the selected designs.

    Note: You cannot unlock a design locked by another user. When you try to unlock a design locked by another user, an error message is displayed indicating that you cannot unlock the design and also displays the user name of the person who has locked the design.

    The following icons in the Lock Status column in ENOVIA Open or ENOVIA Insert dialog box shows the current lock status of the designs:

    • . Indicates that the design is locked.
    • . Indicates that the design is unlocked.

The selected designs are locked or unlocked depending on your choice.
