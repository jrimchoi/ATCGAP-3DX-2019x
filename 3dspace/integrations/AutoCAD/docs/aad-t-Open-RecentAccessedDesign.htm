Viewing Recently Accessed Designs

You can use this tab to view designs accessed during a selected date range.

Related Topics
Opening Designs From ENOVIA

  1. Click ENOVIA > Open or Insert from AutoCAD toolbar.

    The ENOVIA Open or Insert dialog box opens.

  2. Click the tab.

    The left frame displays the different date ranges available.

  3. Click a date range in the left frame.

    The designs accessed in the selected date range is displayed in the right frame. For example, if you select Last 7 Days in the left frame, then the designs accessed during those days are listed.

  4. Select one or more designs and right-click to view the available options such as Open or Insert, Lock, Unlock, Open Version, Baselines, Explore in ENOVIA, and Properties.

    Note: If multiple revisions of the same design is checked out to the same directory, then only the last checked out revision is displayed under the Recently Accessed Design tab. However, if the user checks out multiple revisions of the same design to different directories, then the last checked out design in each of these directories is displayed under the Recently Accessed Design tab.
