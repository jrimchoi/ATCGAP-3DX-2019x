Creating Designs Using Templates

This task shows you how to create designs using predefined templates.

Before you begin: Required templates must exist. Templates are created by the Integration Administrator or an user who has appropriate access privileges. See ENOVIA Designer Central Installation and Administrator's Guide for details on creating templates.

  1. Click ENOVIA > New from AutoCAD toolbar to open the ENOVIA New dialog box.



  2. Enter details for the new design including the following:

    • Templates. Select a template listed for the ACAD design to be created. The Attribute section lists the attributes of the selected template.

      Specify the Name of the design using one of the following methods:

    • Name. Enter a name for the design.
    • Autoname. Select the Autoname check box and select the naming series from the drop down list. Selecting the Autoname checkbox disables the Name field.
    • The attributes that are mapped by your Integration Administrator are listed in the Attributes section. Enter values for the attributes under the Value column.

  3. Click OK when done.

    The template file is downloaded to the Checkout directory defined in the Preferences and opened in AutoCAD as the active session.

    Note: The design is not saved to ENOVIA until you save the design using ENOVIA > Save option.

The new design is created with the selected template and is loaded in AutoCAD.
