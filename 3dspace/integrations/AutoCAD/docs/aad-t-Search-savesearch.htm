Saving Your Search

This task shows you how to save a query for future use. If you build a query that you need to run periodically, you can save it to have it available for future use.

Before you begin: The search query must be defined.

Removing Saved Search

  1. Click in ENOVIA Open dialog box.

    The Save Query dialog box opens.

  2. Enter the following:


    • Name. Enter a name for the query.
    • Description. Enter the description for the query.

  3. Click OK when done.

The search is saved with the specified name and is listed in the Saved Queries tab in ENOVIA Open diaog box.
