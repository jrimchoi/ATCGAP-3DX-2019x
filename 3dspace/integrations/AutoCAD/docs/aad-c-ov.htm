ENOVIA Collaborative Design for AutoCAD in a Nutshell

ENOVIA Collaborative Design for AutoCAD provides a multi-site AutoCAD design data management solution for the extended enterprise. It allows designers to access and share each other's designs from within the native AutoCAD user interface by leveraging the design team collaboration capabilities of ENOVIA Designer Central.

ENOVIA Collaborative Design for AutoCAD puts access to ENOVIA capabilities in the AutoCAD user interface, allowing designers to effortlessly access, manage, share, and store CAD design data without leaving their preferred environment. ENOVIA Collaborative Design for AutoCAD facilitates process workflow, increases data integrity, and improves configuration management.

From within ENOVIA Collaborative Design for AutoCAD, users connect to the database using Designer Central engine predefined access setup to search and browse, lock and check-out designs including associated drawings to their local drives. After modification, users again utilize capabilities provided by ENOVIA Designer Central to store changes in the ENOVIA database.
