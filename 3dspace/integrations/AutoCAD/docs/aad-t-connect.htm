Connecting to ENOVIA From AutoCAD

This task shows you how to connect from AutoCAD to ENOVIA.

Before you begin:
  • The AutoCAD User role must be assigned to you by the Administrator. For more details, see ENOVIA Collaborative Design for AutoCAD Installation and Administrator's Guide.
  • User should be assigned with ACAD license. For more details, see "Assigning Licenses" section of ENOVIA Collaborative Design for AutoCAD Installation and Administrator's Guide.

  1. Click ENOVIA > Login in the AutoCAD tool bar.

    Note: AutoCAD connects to ENOVIA if an active session exists.

    ENOVIA Login dialog box opens.

  2. Enter your User Name and Password and click Login.

    You are logged in to ENOVIA. To check whether your log in is successful, click ENOVIA from AutoCAD toolbar. All ENOVIA commands except Login are enabled

    Note: The system tray displays the icon when connected.

You are logged in to ENOVIA database.
