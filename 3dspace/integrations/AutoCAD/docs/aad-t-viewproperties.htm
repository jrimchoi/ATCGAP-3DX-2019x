Viewing Properties

This task shows you how to view properties of a design that exists in ENOVIA which is active in SolidEdge.

Before you begin: The design must be active in the SolidEdge session and exist in ENOVIA database.
Related Topics
Viewing Design Details in ENOVIA Designer Central

  1. Click ENOVIA > Properties from SolidEdge toolbar.

    The ENOVIA Properties dialog box opens.

    For the design, the ENOVIA Properties dialog box lists the type, name, revision, description, state, owner, policy, and vault. It also lists,

    • Title. The title entered for the design.
    • Locked By. User name of the person who has locked the design.
    • Checkin Comment. Comments entered when the design is saved to ENOVIA.
    • Renamed From. Last used name of the design if the design has been renamed.

  2. Click OK.

    The ENOVIA Properties dialog box closes.

The ENOVIA Properties dialog box lists details of the design in ENOVIA.
