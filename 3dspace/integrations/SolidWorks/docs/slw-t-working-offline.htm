Working Off-line

You can work on files in your local cache off-line when you are not connected to an ENOVIA vault.

In off-line mode, only files in your local cache are accessible. Files that are currently checked out are writeable, but you must return to working onlne before you can check in changes.

  1. Do one of the following:

    • When logging in from SolidWorks or Windows Explorer, in the ENOVIA Login dialog box, enter your User name and Password and click Work Offline.
    • In Windows Explorer, in the ENOVIA Collaborative Design for SolidWorks menu bar, click Tools > Work Off-line.

    A notification indicates that the vault is going off-line.

    In Windows Explorer, the folders in the vault display as blue when the vault is off-line.

    When working in SolidWorks, the commands on the ENOVIA menu, CommandManager and task pane toolbar are unavailable. The columns that normally provide status information are blank because you are not connected to the database and the preview pane indicates that you are working on a local copy of the file that is outside the file vault.

  2. Work with files in the local cache.

  3. To return to working on-line:

    1. If you have files from the local view of the vault open in SolidWorks, save your changes and close them.
    2. In Windows Explorer, navigate to the local view of the vault.
    3. In the ENOVIA menus above the right pane, click Tools > Work On-line.

    Folders no longer display as blue and you can use revision control commands again.
