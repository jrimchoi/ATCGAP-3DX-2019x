Setting Folder Properties

If you have access rights, you can modify folder properties by displaying the folder's Properties dialog box and editing the Folder Data Card tab.

The Folder Data Card tab shows values of data card variables, such as the Project Name, Description, and Author.


  • The View Values tab displays the values.
  • The Edit Values tab lets you change the values if you have access rights.

  1. Right-click a folder and click Properties.

  2. In the Folder Data Card tab, select the Edit Values tab.

  3. Enter changes and click Apply.

  4. Click OK.

  5. To apply the changes to files already created in this folder, select the folder and click Modify > Update > Values in files.
