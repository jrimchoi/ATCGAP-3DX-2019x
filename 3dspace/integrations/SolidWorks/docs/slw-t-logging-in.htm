Connecting to ENOVIA

To manage SolidWorks files in ENOVIA Collaborative Design for SolidWorks, you log in to the ENOVIA vault. You can log in from within the SolidWorks application or from Windows Explorer.

This task shows you how to:

  • Connecting from SolidWorks
  • Connecting from Windows Explorer

Connecting from SolidWorks

When you open or save a file in SolidWorks to an ENOVIA vault, if you are not already logged in to the vault, you are prompted to log in.

The Administrator must have assigned the SolidWorks User role to you. For more details, see the ENOVIA Collaborative Design for SolidWorks Installation and Administratorâ€™s Guide.

  1. In SolidWorks, do one of the following:

    • Click File > Save As.
    • Click File > Open.

    The ENOVIA login dialog box appears.

  2. Type your User name and Password and click Log In.

    Tip: The log in screen has a Work Offline button that lets you work on files in your local cache off-line when you are not connected to an ENOVIA vault. See Working Off-line.

    After you log in, the ENOVIA icon appears in the notification area at the far right of the taskbar. It is present until you exit ENOVIA Collaborative Design for SolidWorks.

You are logged in to ENOVIA Collaborative Design for SolidWorks.

The notification area at the far right of the task bar displays the icon when you are connected. The icon also appears on the ENOVIA tab of the task pane.

When you open a document from within SolidWorks, the ENOVIA menu and the CommandManager are available.

Connecting from Windows Explorer

When you do not have SolidWorks running, you can log in to the vault from Windows Explorer.

  1. Start Windows Explorer and navigate to the location where you created the vault view.

    Vaults are displayed with the ENOVIA icon next to the vault name.

  2. Double-click the vault icon.

    The ENOVIA login dialog box appears.

  3. Type your User name and Password and click Log In.

    The ENOVIA icon appears in the notification area at the far right of the taskbar.
