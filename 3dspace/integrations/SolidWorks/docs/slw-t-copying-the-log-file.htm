Copying the Local Log File

You can copy your local ENOVIA Collaborative Design for SolidWorks log file to a .cog file to make it available to Support or your administrator.

To copy the log file:

Related Topics:

Log File Dialog Box

  1. Open the log file.

  2. Click Copy log file (tool bar).

  3. To save the file, paste it:

    • On the desktop
    • In a folder
    • As an attachment to an email or other document
