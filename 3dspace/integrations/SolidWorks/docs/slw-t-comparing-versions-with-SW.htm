Comparing Versions Using SolidWorks Compare Documents

You can use the SolidWorks Compare Documents command to compare files in a vault or to compare a vault file with a file outside the vault.

See SolidWorks Help: SolidWorks Utilities: Compare Documents.

  1. In SolidWorks CommandManager, click Office Products > SolidWorks Office > SolidWorks Utilities.

  2. Click Compare Documents.

  3. Under Version 1 and Version 2, select the versions to compare.

    The selected versions open in separate windows.

  4. Use the tools in the Compare Documents pane to complete the comparison.
