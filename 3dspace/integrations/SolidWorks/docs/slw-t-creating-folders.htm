Creating Folders

In the local vault view in Windows Explorer, you create folders in workspaces to store your vault files.

Before you begin: To create a folder, you must first create a workspace. See Creating a Workspace and Folder.

  1. In the navigation pane, click on a workspace to open it in the right pane.

  2. Right-click and select New > Folder.

  3. Name the folder.

    The folder is automatically saved to the vault.
