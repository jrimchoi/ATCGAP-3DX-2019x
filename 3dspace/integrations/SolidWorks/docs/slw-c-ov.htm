ENOVIA Collaborative Design for Solidworks in a Nutshell

ENOVIA Collaborative Design for SolidWorks provides a multi-site SolidWorks design data management solution for the extended enterprise. It enables designers to access and share each other's designs from within the native SolidWorks user interface by using the design team collaboration capabilities of ENOVIA Designer Central.

ENOVIA Collaborative Design for SolidWorks provides access to ENOVIA capabilities within the SolidWorks user interface, enabling designers to access, manage, share, and store CAD design data effortlessly, without leaving their preferred environment. ENOVIA Collaborative Design for SolidWorks facilitates process workflow, increases data integrity, and improves configuration management.

From within ENOVIA Collaborative Design for SolidWorks, you connect to the database using Designer Central's predefined access setup. You can get and check out designs and associated drawings to your local drives. After you make your desired modifications, you use ENOVIA Designer Central to store changes in the ENOVIA database by releasing and checking the files back in.

Getting the Most Out of this Guide

To get the most out of this guide, we suggest that you read and perform the step-by-step user tasks, which cover all product functions.

Getting Started describes common tasks that are used each time you use ENOVIA Collaborative Design for SolidWorks.

Using the ENOVIA User Interface in SolidWorks describes the user interface, both within the SolidWorks application and in Windows Explorer.

Adding Files to the ENOVIA Database describes how to add files to the ENOVIA database when you save them from SolidWorks and by using Windows Explorer copy and paste, and drag and drop operations.

Editing Designs from the ENOVIA Database describes how to check documents out of the ENOVIA database so that you can edit them in SolidWorks.

Viewing and Editing File Properties from Windows Explorer describes how to view the properties of a design in the ENOVIA database from within SolidWorks and edit properties if you have the design checked out.

Using Workspaces describes how to manage designs using the Integration Exchange Framework Client.

Viewing Design Details in ENOVIA Designer Central describes how to view the CAD Portal View page of the design from SolidWorks.

Files and Folders describes how to create folders and edit file and folder properties.

Versions and Revisions describes how to get specific versions of files, view files history, compare file versions, and increment revisions.
