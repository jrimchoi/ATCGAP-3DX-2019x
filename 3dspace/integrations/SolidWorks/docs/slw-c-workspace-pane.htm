The Workspace Pane

In the File Explorer integration with Windows Explorer, the first level of organization is the Workspace.

Related Topics:
Creating a Workspace and Folder
Using the ENOVIA CommandManager to Manage Workspaces


Workspaces must be created using Windows Explorer, although you can open Windows Explorer to a specific workspace from within SolidWorks. See Using the ENOVIA CommandManager to Manage Workspaces. You can also access a specific workspace by browsing to it in Windows Explorer.



The left side of the workspace pane contains the folders you have created to contain the documents you add to the vault. See Creating Folders.

The right side of the pane displays:


  • High level properties for the workspace.

    You can also see this information using the Designer Central Web Client.

    Properties include:

    • The workspace image present in Designer Central
    • The workspace life cycle state
    • The date the workspace was created
    • The buyer desk responsible for the design
    • A description of the workspace.

  • Contact information for the workspace owner, including:
    • The owner's name
    • The last time the owner logged in
    • The owner's work location, telephone number, cell phone number, and email address.
