ENOVIA Collaborative Design for SolidWorks Menu Bar

In Windows Explorer, the ENOVIA Collaborative Design for SolidWorks commands are divided into four menus for ease of use.

The tasks described are those that are specific to ENOVIA Collaborative Design for SolidWorks.

  • Actions Menu
  • Modify Menu
  • Display Menu
  • Tools Menu

Actions Menu

Check Out

Checks out the file for editing.

See Checking In Files.

Check In

Checks the selected topics into the vault.

See Checking In Files.

Get Latest Version

Copies the latest version of the file from the archive server to your local vault.

See Getting Versions.

Get Version

Displays a numbered list of previous versions of the file for you to select from.

See Getting Versions.

Undo Check out

Cancels a checkout without saving changes.

See Undoing a Check Out.

Modify Menu



 Cut Copy Paste

Pastes copied files or folders into the current folder.

The copied files are checked out to you and are not available to other users until you check them in.

Display Menu

History

Displays history of file or folder.

See Viewing History.

Properties

Displays properties of selected file or folder.

See Viewing and Editing File Properties from Windows Explorer.

Explore in ENOVIA

Opens the ENOVIA Collaborative Design for SolidWorks Web Client.

See Viewing Design Details in ENOVIA Designer Central.

Tools Menu

Clear Local Cache

Removes the whole vault view from Windows Explorer and all local cache files and folders. Check in all files and log out of the vault before deleting the vault view.

See Cached Files.

Work Off-line

Lets you work on files in your local cache off-line when you are not connected to an ENOVIA vault.

See Working Off-line.
