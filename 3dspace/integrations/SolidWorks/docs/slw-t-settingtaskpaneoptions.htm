Setting Task Pane Options

You can set display options to modify the appearance of the ENOVIA task pane.

  1. To set task pane options, on the main task bar, click ENOVIA > Options.

  2. On the Server tab, select options that specify the messages and prompts that appear, and how the tree structure in the upper task pane is displayed.

  3. On the View Settings tab, select settings to control how the information in the task pane is displayed. For example, you can:

    • Control how the arrow keys affect file selection.
      Tip: You can choose to have arrow buttons jump to the next file in the tree, rather than to the next row.
    • Select whether to display instances of parts or assemblies.
    • Select the file information to display.
      Tip: You can choose whether to display files or variables, revision, version, who has the file locked, the lifecycle state, quick child information, and configuration information, among other options.
    • Select the format of displayed information.
      Tip: You can choose whether to display information as a node in a tree, in the preview, in a column of its own, or as part of the file name column.
    • Specify whether the quick-info icon displays.
    • Specify changes in the background colors, selecting which colors indicate various states.

  4. Click OK.
