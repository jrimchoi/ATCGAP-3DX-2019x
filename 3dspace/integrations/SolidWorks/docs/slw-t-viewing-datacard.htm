Viewing and Editing a Data Card in SolidWorks

You can view a design's data card from within SolidWorks.

You can modify the design's custom properties by using the SolidWorks File > Properties command.

Before you begin: To be able to modify editable fields of the data card, check out the file before you view it.
Related Topics:
Viewing and Editing File Properties from Windows Explorer
File Properties Dialog Box

  1. In SolidWorks, click ENOVIA > Show Card

  2. In the Properties dialog box, if there are multiple configurations, select the configuration to view or edit.

  3. For fields that are editable, you can add or modify values.

    Note: Many fields are controlled by variables and cannot be edited.

  4. Click the configuration's Description tab to add a description of your changes.

  5. Click OK.
