ENOVIA Task Bar Menu

The ENOVIA menu lets you access help, view a log file of events on your local system, browse to the SolidWorks Web site, and exit ENOVIA Collaborative Design for SolidWorks.

Click the ENOVIA icon in the notification area at the far right of the task pane to display the menu.



 Help

Displays the ENOVIA Collaborative Design for SolidWorks Help.

View Log File

Opens the Log File dialog box so that you can view a log file of events on the local client.

See About the ENOVIA Log File

Exit

Closes the connection to ENOVIA Collaborative Design for SolidWorks.

See Disconnecting from ENOVIA.
