Incrementing Revisions

A file that reaches a certain state (for example, an assembly is approved for manufacturing) can be assigned a new revision number. You manually increment the revision number when you check a file into the vault.

Before you begin: To increment revisions manually, you must have appropriate permissions.

  1. Do one of the following:

    • When working in SolidWorks, to increment the revision of the active design, click ENOVIA > Check In or click Check in on the ENOVIA CommandManager.

      To increment the revision of a referenced file, in the task pane, select the file and click Check In .

    • In the local vault view in Windows Explorer, select the files and click Action > Check In.

    The Check In dialog box opens, showing the current revision number in the Local Ver. and New Ver. columns.

  2. Verify that the files whose revision should be incremented can be checked in.

    If OK is not displayed in the Result column, a message explains the problem (lack of access rights, for example). The administrator must give you rights to increment revisions for the file.

  3. To increment a file's revision, check the box in the Inc. Rev column.

    The value in the New Ver. column shows the change in the revision.

  4. Click OK.

  5. To verify the new revision number, select the file and click Display > History.
