What's New?

This section provides information about the new and enhanced features in the 2012 release.

New Features

Release 2012 provides several new features and commands that enhance your ability to locate and work on files.

New Commands
The ENOVIA menu, ENOVIA task pane, and ENOVIA CommandManager all contain new commands. For instance, you use the new Check In and Check Out commands, respectively, to check out an active file, and check files into the vault so that other users can edit them.

Other new commands enable you to view file attributes such as file history, and get a particular version, or the latest version, of a file for editing.

You can execute the SolidWorks commands File > Open and File > Save to act on files you select in your vault, and you can invoke various SolidWorks commands on the SolidWorks Insert menu.

Enhancements in Navigation to Files

In Release 2012, when you log in to a vault, the user interface is integrated with Windows Explorer, with various menu options, toolbar buttons, and dialog boxes. The Manage ENOVIA Workspace, Explore in ENOVIA, and Explore in Windows commands in the ENOVIA CommandManager all include enhancements in Release 2012.


  • The Manage ENOVIA Workspace command opens the local vault view of Windows Explorer to the workspace that contains the folder where the active SolidWorks design resides.
  • The Explore in ENOVIA command enables you to login to the ENOVIA Designer Central server and act from there on files in your vault.
  • The Explore in Windows command opens the local vault view of Windows Explorer to the folder containing the active design. There is no need to navigate to the design.
SolidWorks 2011 Support
This release supports SolidWorks 2011.
