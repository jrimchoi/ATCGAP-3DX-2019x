File is Open in Another Application

ENOVIA Collaborative Design for SolidWorks operations can fail when a file is open in another application. Close the file in the other application and try the operation again.

Tip: If the file is open in SolidWorks and the ENOIVA add-in is installed, you can operate on the file from the add-in while the file is open.

Checking In and Checking Out
The Check In and Check Out dialog boxes display The file is open in another application and block the operation.
Editing a Data Card
You cannot save data card changes if the file is open in another application.
