Using the ENOVIA Task Pane

You can use the ENOVIA task pane to execute commands and view information about files in the database.

Related Topics:
Setting Task Pane Options

  1. Open a file in the ENOVIA database.

  2. Click the ENOVIA icon to display the ENOVIA task pane.

    The upper section of the task pane lists the open file and its components. Columns provide details about the listed files.

    The lower section displays the same file details graphically, letting you see all the file information when you need to keep the task pane small so that there is more room to work in the work area.

  3. Use the toolbar buttons at the top of the task pane to perform ENOVIA and SolidWorks commands on the selected file.



    Note: You can also right-click to access commands.

    Button Command Description


    		Get Version Gets the latest version of the file in the vault.


    		Get Latest Version Lets you get a specific version of the file from the vault.


    		Check Out Checks the selected file out. Other users are prevented from modifying the file.


    		Check In Checks the selected file in.


    		Show Card Displays the properties dialog box.


    		Explore in ENOVIA Opens the Web Client.


    		Explore in Windows Opens the File Explorer window.


    		Zoom to Selection SolidWorks command. Zooms to the selected part or assembly.


    		Open SolidWorks command. Opens the selected part or assembly.


    		Edit SolidWorks command. Lets you edit the selected part or assembly.

  4. In an assembly or multibody part, select a component to highlight it in the work area.
