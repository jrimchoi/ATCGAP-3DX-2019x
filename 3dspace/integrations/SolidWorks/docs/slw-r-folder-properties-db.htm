Folder Properties Dialog Box

You can modify data card values in the Folder Properties dialog box if you have the right permissions in your user profile. The Folder Properties dialog box appears when you right-click the folder and select Properties.



General tab

Lists the folder name, type, and location.

Folder Data Card tab

Shows values of data card variables, such as the State, Owner, and Vault.

If the folder is checked out, you can edit the value of the Description field.
