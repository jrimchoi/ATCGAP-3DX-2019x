Undoing a Check Out

You can cancel a check out without saving changes.

  1. Do one of the following:

    • To undo the check out of the active design in SolidWorks, click ENOVIA > Undo Check Out.
    • In the local vault view in Windows Explorer, right-click a file or folder you no longer want checked out and click Undo Check Out.

  2. In the Undo Check Out dialog box, set column values as needed.

    See Undo Check Out Dialog Box.

    Important: If you undo a check out after you have made changes to the file, the changes will be lost.
