Versions and Revisions

ENOVIA Collaborative Design for SolidWorks maintains the history of files, including all versions and revisions. Each time you modify or check in a file, a new version is created. You can view, retrieve, compare, and restore earlier versions of files. When a file reaches a certain state (an assembly is approved for manufacture, for example), its revision can be incremented, either manually or as part of an automated lifecycle.
