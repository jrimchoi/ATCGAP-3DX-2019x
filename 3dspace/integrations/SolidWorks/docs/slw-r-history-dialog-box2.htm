History Dialog Box

Use the History dialog box to view the version and workflow history for a selected file.

  • Event List
  • Details
  • Buttons

Related Topics:
Viewing History

To view the History dialog box when you are working in SolidWorks, click History in the CommandManager.

To view the History dialog box when you are managing vault files in Windows Explorer, select a file and click Display > History or History (toolbar).



Event List

Event Event name, such as Created and Checked in.
Version The revision and version number for the event.
User User who performed the event.
Date Date and time of the event.
Comment Comment, if any, associated with the event.

Details

DIsplays details for the selected event, including the file name. All other fields repeat the details found in the columns of the event list. All fields are read-only.

Buttons

View Displays a preview of the file.
Get Retrieves selected version to local cache.
Save Saves the selected version to file name you specify.
Print Prints the event list. Select print options from the Print dialog box, then click OK.
Help Displays this help page.
Close Closes the dialog box.
