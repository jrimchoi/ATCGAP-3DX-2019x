Displaying Event Details

Use log event details to troubleshoot problems with ENOVIA Collaborative Design for SolidWorks.

Log File Dialog Box

  1. Open the log file.

  2. In the log file dialog box, double-click the event.

    A dialog box displays the following:

    • A description of the event
    • The date the event occurred
    • The module and function involved
    • The event error code
    • A description of the error code

  3. To view the previous event in the log, click Prev.

  4. To view the next event in the log, click Next.

  5. To copy the event details, click Copy.

    You can paste the content into an email or document.
