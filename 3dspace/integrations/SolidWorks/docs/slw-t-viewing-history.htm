Viewing History

You can view the version history for changes made to a file by users in your enterprise.

  1. Do one of the following:

    • In SolidWorks, on the ENOVIA CommandManager, click History.
    • In the local vault view in Windows Explorer, select the file and click Display > History or History (toolbar).

  2. Select a version of the document to display its details below.

From the History dialog box, you can View a version of a file, Get the selected version to copy it to your local cache, and Save a version to a specified location and file name.
