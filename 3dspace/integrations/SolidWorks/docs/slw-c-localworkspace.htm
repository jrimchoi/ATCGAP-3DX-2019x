About Managing Workspaces

A workspace is a collection of folders that contains objects, documents, and other information you can use while working with ENOVIA Collaborative Design for SolidWorks.

Related Topics:
Using the ENOVIA CommandManager to Manage Workspaces

The ENOVIA Integration Exchange Framework Client enables you to use the Manage Workspace feature to view and manage files stored in the ENOVIA database.

Note: The ENOVIA Integration Exchange Framework Client must be installed for you to use this feature.
