ENOVIA CommandManager

Like the ENOVIA menu, the commands in the ENOVIA CommandManager operate on the currently active model in the graphics area.

Additional commands let you work with the document in Designer Central or the File Explorer,



 Check Out

Lets you check out the active file.

Other users can view the files but they are prevented from editing them.

See Checking Out Files

Check In...

Lets you check files into the vault so that they are available to other users for editing.

See Checking In Files

Undo Check Out...

Cancels a checkout without saving changes.

See Undoing a Check Out

Get Latest Version

Copies the latest version of the file from the archive server to your local vault.

See Getting Versions

Get Version

Displays a numbered list of previous versions of the file for you to select from.

See slw-t-getting-file-versions.htm#slw-t-getting-file-versions

Show Card...

Displays the file's datacard.

History

Displays the History dialog box.

See Viewing History

Explore in ENOVIA

Lets you log in to the ENOVIA Designer Central server. The summary information for the file is displayed.

See Viewing Design Details in ENOVIA Designer Central

Explore in Windows

Opens a Windows Explorer window to the folder that contains the model.

Manage ENOVIA workspace

Opens a Windows Explorer window to the workspace that contains the model.

See Using the ENOVIA CommandManager to Manage Workspaces
