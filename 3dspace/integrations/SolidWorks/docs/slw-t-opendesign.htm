Opening a Particular Version of a Design

From within SolidWorks, you can open an earlier or later version of a design from the ENOVIA database to continue development of an existing design.

Before you begin: Multiple versions of the design must exist in the ENOVIA vault.

  1. In a File Explorer window, browse for the file you want to open and click Get Version.

    See About Getting File Versions

    The file downloads to your local cache.

  2. Invoke the SolidWorks File > Open command.

The selected version of the design opens in SolidWorks.
