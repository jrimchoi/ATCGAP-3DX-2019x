Log File Dialog Box

You can view error events reported by this ENOVIA Collaborative Design for SolidWorks client.

The Log File dialog box appears when you double-click the icon in the navigation area to the right of the task bar and click View Log File.



The title bar shows the log file location. The defaults are:

Windows 7 and Vista C:\Users\<user name>\Local\Dassault Systems\ENOVIA\ENOVIA Collaborative Design for SolidWorks\ENOVIA.cog
Windows XP Documents and Settings\<user name>\Local Settings\Application Data\Dassault Systems\ENOVIA\ENOVIA Collaborative Design for SolidWorks\ENOVIA.cog

Where <user name> is the Windows logon user name on this computer.

An ENOVIA Collaborative Design for SolidWorks log file has the .cog extension.

Log Summary

Displays a summary of all log entries.

Double-click an entry to display the details for that entry.

Show full log

Shows all log entries as text.

Error events are highlighted.
Refresh

Refreshes the log.
Copy log file

Copies the log file object to the Windows clipboard.

You can paste a copied log file into a folder or directly into an email as an attachment.

You can also copy the log file by right-clicking Log File in the left pane and selecting Copy.
Empty the log

Deletes the client log.

A new empty log is created.
Find Next

Searches the log for the next entry that has the text specified in the Search text: field.
  Search text To search for error messages with specific text, type the text in the Search text field and click Enter.

Click Enter again to move to the next message that contains the search text.
