Clearing the Local Cache in the Working Folder

Clearing the local cache removes all files that are not checked out from the working folder.

Related Topics:
Cached Files
Clearing the Local Cache During Check-in

  1. To clear the local cache in Windows Explorer, do one of the following:

    • Select any folder other than the vault root and click Tools > Clear Local Cache.
    • Right-click the vault root folder and click Clear Local Cache.

  2. In the Clear Local Cache dialog box, to remove Toolbox files, clear Do not remove Toolbox files.

    By default, Toolbox files are not cleared from the cache.

  3. Click Yes.
