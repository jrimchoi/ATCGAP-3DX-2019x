About Opening Designs from the ENOVIA Database

This topic describes how files in the ENOVIA database are handled when you open them in SolidWorks.

If you navigate to a file in Windows Explorer, you can specify the ENOVIA commands Get or Get Version to retrieve a file that has been updated by another user so that you can modify the design in SolidWorks.

See About Getting File Versions.

About Opening Designs with Attachments

Attachments are SolidWorks-referenced files that are embedded or linked to a SolidWorks design. Since only SolidWorks files can be stored in an ENOVIA vault, you can open only attachments that are referenced SolidWorks parts, drawings, and assemblies.

When you open a file with a linked part, assembly, or drawing, the attachment is also opened along with the model.

Opening Multiple Designs

In SolidWorks, you can select only one file to open at a time. However, if the file references other SolidWorks parts, assemblies, or drawings, they also are opened.

Selected designs are opened to the working directory and are loaded to the active session.
