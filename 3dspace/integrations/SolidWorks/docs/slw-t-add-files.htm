Adding Files to the Vault Using Windows Explorer

If you are not working in SolidWorks, you can add files to the vault using standard Windows Explorer copy and paste or drag and drop functions.

Before you begin: In Windows Explorer, create the workspace and folder to contain the files.

See Creating a Workspace and Folder.

  1. Navigate to the location of the files to be added and select them.

  2. Do one of the following:

    • Copy the files, navigate to the folder that will contain the files, and paste the files.
    • Drag the files to the folder that will contain them.

  3. In the local vault view folder, select the files.

  4. Check the files into the vault.
