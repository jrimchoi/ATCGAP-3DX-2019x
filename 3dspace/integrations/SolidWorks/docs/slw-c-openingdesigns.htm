Opening Designs from the ENOVIA Database

Opening a design copies it from the ENOVIA database to the local view of the vault on your computer.

From within SolidWorks, the options to open a file from a vault in the ENOVIA database are:


  • File > Open. Use this option to open designs from the ENOVIA database.
  • File > Insert. Use this option to insert a design from the ENOVIA database in the currently active design in SolidWorks.
