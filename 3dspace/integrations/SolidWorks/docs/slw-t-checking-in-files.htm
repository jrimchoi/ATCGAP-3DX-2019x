Checking In Files

You check in files after editing to make them available to users with appropriate rights. Until you check in a file, changes are saved in your local version only.

Note: The new version of each checked in file is associated with the current versions of its references. When this file version is accessed, all reference versions are retrieved as if they were attached.

  1. Do one of the following:

    • In SolidWorks, click ENOVIA > Check In.
    • In the local vault view in Windows Explorer, right-click a file or folder and click Check In.

    The Check In dialog box opens, listing the files that can be checked in. Columns provide file information and checkin options.

    See Check In Dialog Box.

  2. In the Check In column, select the files to check in.

  3. You can also do one or more of the following:

    • In the Keep Checked Out column, select files you want to keep checked out for further editing.
    • In the Inc. Rev. column, select files whose revision you want to have incremented upon checkin.

    The options you selected are applied when the file is checked into the vault in the ENOVIA database.

  4. Enter a comment to identify the version.

  5. Click Check in.
