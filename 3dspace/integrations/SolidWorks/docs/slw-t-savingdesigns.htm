Saving Designs to a Vault in the ENOVIA Database

Saving a design to the ENOVIA database is a two-part process. First, you use the SolidWorks File > Save or File > Save As command to save the file on your local machine. Then you check the file into an existing ENOVIA vault.

This task shows you how to:

  • Saving New Designs to the ENOVIA Database

Saving New Designs to the ENOVIA Database

You can save a new design that is open in SolidWorks to a workspace and folder in your local view of the ENOVIA database.


  • You must rebuild the assembly before you save it to the ENOVIA database.
  • The workspace and folder must already exist in the ENOVIA database.

  1. From the SolidWorks toolbar, click File > Save As.

  2. In the Save as dialog box, navigate to an existing folder in the local view of the vault.

  3. Click Save.

    The file is saved to your local cache.

  4. Click ENOVIA > Check In.

    You can also:

    • Click Check In in the ENOVIA CommandManager or ENOVIA task pane toolbar.
    • Right-click in the FeatureManager Design Tree and click Check In.

    The Check In dialog box opens, listing the files that can be checked in. Columns provide file information and checkin options.

    See Check In Dialog Box.

  5. Select one or more files to check in and set column values as needed.

    See Checking In Files.

  6. Enter a comment to identify the version.

  7. Click Check In.
