Creating a Workspace and Folder

You create workspaces in the local view of the vault to contain the folders that organize your vault files.

  1. In Windows Explorer, navigate to the location where you created the local view of the vault during installation.

  2. In the right pane, right-click and click New > Workspace.

  3. Type a name for the workspace.

  4. Double-click the workspace to open it.

  5. Right-click in the workspace and click New > Folder.

  6. Type a name for the folder.

You can now add files to the vault by:

  • In SolidWorks, saving files to the folder
  • In Windows Explorer:
    • Copying and pasting files to the folder
    • Dragging and dropping files into the folder
