Command Action Trigger


Contents

The 3dxtools Command Action Trigger is used to connect a command in the ENOVIA live collaboration web interface with a series of 3dxtools Activities to execute when the command is clicked. The 3dxtools Command Action trigger works by getting an ENOVIA command to call the 3dxtools.jsp, passing in a key identifying the 3dxtools Trigger object to use. The 3dxtools.jsp then runs the core 3dxtools passing in the key, the current object, any table selected objects and all the other current URL parameters. Some examples of the operations you can perform using these commands are:

•Use the PDF Report Generator Activity to generate a PDF report.
•Use the Initiate Route Activity to put the current object into a route.
•Use the Export Activity to select and download the metadata and files for multiple objects.

 

Configuration and Settings

To setup a 3dxtools Command Action Trigger follow the steps defined below.

1. Add the Command object in ENOVIA ➣

•In ENOVIA MQL add a new Command object.
•Go to Windows Start Menu ->mql ->click on "Start Mql.exe".
•set the context by writing below query,

  MQL>set context user creator;

•Set the Name, Description and Label values to something suitable.
•The Link for command to be added with following values:

Name

Value

Notes

Href

${ROOT_DIR}/3dxtools/3dxtools.jsp?drtoolsKey={3dxtools Trigger Key}

Where {3dxtools Trigger Key} is the name of the 3dxtools Trigger Action object to find and execute.
•List of Settings to be added:

Name

Value

Notes

Registered Suite

3dxtools

Uses the settings and string resource file in 3dxtools

Submit

true

Submits the current object ID on the command URL

Target Location

hiddenFrame

Hides the JSP from the user
•Query to add new command

Syntax:

    Mql>add command ${Command_Name} Description "description value" lable "Command Label" href "${ROOT_DIR}/3dxtools/3dxtools.jsp?drtoolsKey={3dxtools Trigger Key}"

    setting "${Setting_Key}"  "${Setting_Value}" ;

Example:

    MQL>add command drV6ToolsDeliverableContentDescriptionExportCommand label "Export" description "export command" href "${ROOT_DIR}/3dxtools/3dxtools.jsp?drtoolsKey=ExportDocument"

    setting "Registered Suite" 3dxtools setting   "Submit" true setting "Target Location" hiddenFrame" ;

 

2. Add the new Command to a Menu object in ENOVIA ➣

•In ENOVIA Business add the newly created Command to a suitable Menu

 Mql>list menu ${Menu_Name};

 Example:

    Mql>mod menu ${Menu_Name} add command ${Command_Name};

  Example:

 

•In ENOVIA confirm the menu command has been correctly added.

Mql>Print menu ${Menu_Name} select command;

 Example:

•Remove Command

 MQL>mod menu remove ${Command_Name};
