Start, Stop or Switch View on Processor Instance


Contents

The Start and Stop buttons provides the ability to start and stop the 3dxprocessor instance service

If you select a 3dxprocessor instance in the grid window and click the Start button then the service will be started. Equally you can start the service from the standard windows Local Services window as well. The Stop button stops the selected 3dxprocessor instance.

When a 3dxprocessor instance is running it is possible to switch the current view to look at the actual running instance. This can be very useful if there are any issues and you want to take a closer look as the process runs. This is particularly important when the 3dxprocessor instance is driving 3rd party applications that might be holding up processing jobs (for instance due to licensing issues or unexpected dialogs popping up). To switch the view simply click on the Switch View button. This will take you into the protected desktop session. If you wish to exit this and return to the normal desktop then click the Return now button in the Interactive Services Detection dialog.
