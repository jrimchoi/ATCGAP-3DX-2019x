Show Processor Status


Contents

3dxtools provides a simple mechanism to configure standard icons to be displayed dynamically in the ENOVIA table column and web form fields,basically it will display different icons based on status of queue objects which will processing on 3dxprocessor . It gives the administrator the ability to turn on or off the display controls  using settings of the table column and fields of web form.

Process Status column displays the different icons based on the process status,

Each Corresponding document has connected queue objects and its attribute drlStatus will update dynamically when the 3dxprocessor runs  ,so the different stages of processor will be shown in the form icon to user like,

 Queue Object is Processing.

 Queue Object is Waiting.

Queue Object is Processing Failed.

Queue Object is Processed.

For each of this icon , hyperlink (click-able) has been given ,on click of these icons ,a proper alert message and tool tip is provided

And for the configuration object ,there will be predefined Derived Output  object which has settings from which the alert message and tooltip will be displayed for icons.

Above configuration can also be used for each documents property page ,

on webforms of the document or any other objects webform where Process Status is required to display,

On the webform all the settings which are required to display "Process Status" is defined
