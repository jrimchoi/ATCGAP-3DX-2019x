The Assignment Form


Contents

Once the Route Template is selected the system will check for a DRLTools Route Setting object linked to the Route Template with a Response value set to Start and a Timing value set Before.

 

1. If found then the settings value will be used to drive the process.

 

If the <chooseUsers> setting values are defined then the following form is displayed, which lists the tasks on the route and provides a dropdown box for each allowing the user to either leave the task assigned to the role. Or select a single user from within that role:

i.e.

 <chooseUsers>

         <chooseUsersForAllTasks>true<chooseUsersForAllTasks>

 <chooseUsers>

 

 

2. If DRLTools Route Setting not found then the OOTB route creation drive the process without any assignee change.

 

 

 

The user is provided with two buttons:

Assign: The selected users are assigned to the route and the route created.

 Cancel: The operation is cancelled and the recently created route and tasks are deleted
