if Tag


Contents

The ifPage tag allows you to test to see if the page to be watermarked is a certain size and orientation. If it matches then the watermark elements it contains will be added to the page.

ifPage XML

<ifPage size="" landscape="">
   <watermark/>

   <watermark/>

   <watermark/>
   <if/>
</ifPage>

 

 

Attributes:

Name

Value

Notes

size

{One of the following page sizes: A0, A1, A2, A3, A4, A5, A6, A7, A8, B0, B1, B2, B3, B4, B5, B6, B7, B8, B9, B10, C2 , C3 , C4 , C5 , C6 , D0 , SRA0, SRA1, SRA2, SRA3, SRA4, RA0, RA1, RA2, Letter, Legal, Ledger, Tabloid", Executive, ANSI C, ANSI D, ANSI E}

The standard page size to match

landscape

{true or false}

Set to true if it should be landscape or false if it should be portrait in orientation.

 

 

Example ifPage XML

 

<ifPage size="A3" landscape="true">
   <watermark style="H1" horizontalPosition="middle" verticalPosition="top" alignment="left" height="25" width="400" underlay="false"> 
      <text>A3 sheet</text><value on="user">First Name</value>
   </watermark> 
</ifPage>
