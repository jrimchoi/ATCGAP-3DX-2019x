Export Type Setting


Contents

To each Export Activity it is possible to link multiple 3dxtools Export Type Setting objects. These allow you to define more specific settings for the object types to be exported. Using these objects it is possible to define on a type by type basis the attributes to export. It is also possible to create more complex values using Copy Values Activity configuration by connecting Copy Values Target Attribute objects and  setting the Target Attribute value to the name of value you want to appear in the exported XML metadata.

 

To create an Export Type Setting object:

•Select the Export Type Settings tab on the Export Activity object.
•Click Actions.
•Select Create DRLTools Export Type Setting...

oThis will open the DRLTools Type Setting create form.
