3dxprocessor Job Server Processors


Contents

Job Server Processors do the processing on the objects requested by Queue Objects. Each processor has its own capabilities and configuration requirements and can perform tasks like PDF view file generation (using the View File Generator Processor), import of XML data from external systems (using the Import System Processor) or offline 3dxtools Activities processing (using the Activity Processor).

A Job Server Processor is initiated by a Job Server Triggers, which when the trigger fires starts an instance of the Job Server Processor and passes the Queue Object into it. The Queue Object provides the Processor with a context of what it is running on (for instance the business object details, file name,  requesting user, etc) and the rest of the Job Server Processor configuration requirements are provided in the Job Server Processor configuration settings.

 

To add a Job Server Processor:

•Select the Triggers tab on the Job Server Controller object.
•Click Actions.
•Select Create Job Server Processor...

oThis will bring up the Create form.
•Select the Type Chooser button and select the type of Processor that is required:

•Fill in the rest of the configuration required (depending on the processor selected).
•The Job Server Processors currently available and their links to their configurations settings are defined below:

Job Server Processor

Description

Activity Processor

This processor provides the ability to offline and run as batch any 3dxtools Activities on a business object.

Custom Processor

This processor allows a simple method of plugging in custom Job Server processors into the system.

Derived Output Generator Processor

This processor generates derived output view files (both 2D and 3D) for CATIA Parts, Products and Drawings.

Import System Processor

This processor allows you to import of XML data from external systems.

PDF Thumbnail Processor

This processor generates thumbnail files from PDF stored within ENOVIA.

View File Generator Processor

This processor is designed to generate PDF view files for objects checked into the ENOVIA store.
