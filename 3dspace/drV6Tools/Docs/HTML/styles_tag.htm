Styles Tag


Contents

The styles tag holds a list of style tags which define styles that are to be used later in the document. These get referenced using a unique name.

 

Example styles XML

<styles>
 <style name="bold" font="Arial" size="9" format="bold"/>
 <style name="boldUnderline" font="Arial" size="9" format="bold,underline"/>
 <style name="normal" font="Arial" size="9" format="" default="true"/>
 <style name="large" font="Arial" size="14" format="" />
 <style name="H1" font="Arial" size="14" format="bold" />
 <style name="H2" font="Arial" size="12" format="bold" />
 <style name="small" font="Arial" size="5" format="" />
 <style name="smallbold" font="Arial" size="5" format="bold" />
</styles>
