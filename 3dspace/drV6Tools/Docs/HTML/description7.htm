 3dxtools provides a  mechanism to configure the standard icons to be displayed dynamically in the ENOVIA table column and web form fields. It gives the administrator the ability to turn on or off the display controls  using settings of the table column and fields of web form.

Based on the status of queue objects icon will be changed on the table column ,for example table column "Process Status" on workspace table,

For each Document ,connected Queue Object's Status will be displayed in the form of Icon on "Process Status" Column,

 Queue Object is Processing.

 Queue Object is Waiting.

Queue Object is Processing Failed.

Queue Object is Processed.

Based on the status of queue objects icon will be changed on the web form field ,for example "Process Status" field on document properties page,
