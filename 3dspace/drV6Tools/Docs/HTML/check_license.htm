Check License


Contents

This command will open the page which will provide the details of the current available license of 3dxtools.  From Home page, navigate to Designrule Tools Admin à 3dxtools License à Check License.  This will open the below page.

 This page will contain the following fields.        

SL No

Field Name

Details

1

3dxtools Build Version

This is the build number of the installed 3dxtools

2

License Type

This is the type of license. It can have the following values. Activated/Temp/Fixed version.

 

Temp: This will have 30 days validity and user must request for the new license.

 

Fixed version: This license is valid for the fixed version mentioned in the license. This license cannot be used for different version of 3dxtools.

 

Activated: This is the activated version of license which provided all features of licensed product till the expiry date.

3

Licensed Products

This provides the information of products for which the license is available. Ex: 3dxdesigner, 3dxcomponents, 3dxprocessor, 3dxtools

4

License Expiry Date

Expiry date of the license.

5

Locations Count

Active locations available in the environment. Default value will be 1.

6

Active Users Count

Active users available in the environment.

7

Environment Name

Name of the environment.

8

Environment Type

This the type of environment. It can have the following values: Production/Test/Validations/Development

9

Customer Name

Name of the customer who is requesting for the license.

10

Customer Address

Communication address of customer.

11

Customer Contact Name

Name of point of contact

12

Customer Contact Email Address(es)

Email address of point of contact. It can be multiple separated by comma.

13

VAR Name

Name of VAR

14

VAR Contact Name

Name of point of contact of VAR

15

VAR Contact Email Address(s)

Email address of point of contact of VAR

16

Request Notes

Notes provided by customer for requesting the license.
