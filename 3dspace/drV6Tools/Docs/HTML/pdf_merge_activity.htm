PDF Merge Activity


Contents

The 3dxtools PDF Merge Activity allows PDF documents of all the selected documents to be merged into a  single PDF document. It is configured with XML.

Configuration and Settings

To setup a 3dxtools PDF Merge Activity:

1. Add an PDF Merge Activity➣

•Select the Dependant Activities tab on either a top level 3dxtools Triggers action object or another 3dxtools Activities object.
•Click Actions.
•Select Create DRLTools PDF Merge Activity...
