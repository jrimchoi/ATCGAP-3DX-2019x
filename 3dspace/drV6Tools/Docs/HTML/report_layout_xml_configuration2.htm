Report Layout XML Configuration


Contents

The Report Layout attribute on the PDF Report Generator Activity contains the full definition of the PDF Report in an XML configuration.

The XML is created by creating a root document tag that describes the whole document. The document tag contains styles and pages sub tags. Styles holds a list of style tags that can be used across the report, and pages holds a list of page configurations. A page holds details that define the page and margin sizes as well as any header and footer information. A page also holds a table which can be used to query the database for further information, such as expand statements. These tables can be configured to populate a known amount of data or to dynamically populate data by using if tags and foreach tags.

 

Full list of tags ➣

Tag Name

Notes

document

Root node, contains all other tags.

styles

Contains a list of style tags.

style

Defines a single style.

pages

Contains a list of page tags.

page

Defines a single page in the PDF report. Contains name, unitOfMeasure, lineSpacing, margin, flowText, fixedText, header and footer tags.

name

Defines the name of a page.

size

Defines the size of a page.

unitOfMeasure

Defines the units used by a page.

lineSpacing

Defines the space between lines on a page.

margin

Defines the margin sizes for a page.

flowText

Defines text of an unknown length.

fixedText

Defines text that is of a fixed length.

header

Defines the header for a page. Contains headerImage and headerText tags.

headerImage

Defines a header image.

headerText

Defines header text.

footer

Defines the footer for a page. Contains footerImage and footerText tags.

footerImage

Defines a footer image.

footerText

Defines footer text.

tables

Contains a list of table tags.

table

Defines a table in the report.  Contains cell, if and foreach tags.

cell

Defines a cell in a table.

foreach

Defines a query that will populate the document with values for each value        returned.

if

Defines an if condition that will only populate a cell if the condition is met.
