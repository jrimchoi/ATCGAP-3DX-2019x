page Tag


Contents

The page tag holds the information for an individual page in the report. This includes the name of the page, the size, header and footers and tables. It is worth noting that a page configuration can span multiple physical pages if the data returned covers multiple pages. A good example of when to use multiple page tags would be for a multi-page document that needs a cover page some central data pages and a closing page that have different styles. The first page would represent the cover page, the second page represent all pages that contained data, and the third page represent the closing page.

 

Example page XML

<page>
 <name>Page1</name>
 <size>A4</size>
 <unitOfMeasure>MM</unitOfMeasure>
 <lineSpacing>1</lineSpacing>
 <margin>
         <top>5</top>
         <left>20</left>
         <right>20</right>
         <bottom>10</bottom>
         <headerHeight>55</headerHeight>
         <footerHeight>20</footerHeight>
 </margin>
 <flowtext>
 </flowtext>
 <fixedtext>
 </fixedtext>
 <header>
 </header>
 <footer>    
 </footer>
 <tables>
         <table>

                 <cell/>

                 <cell/>

                 <cell/>

                 <cell/>
         </table>
 </tables>
</page>
