Webapps Changes


Contents

 This section describe the JSPs and web.xml changes required to enable 3dxtools routes enhancement functionality.

 

 

Prerequisites:

 3dxtools is installed and application is deployed.

 

Post Configuration:

 Restart the ENOVIA services.

 

Changes:

Location

../components/

  ../common/

Take backup of emxLifecycleTasksMassApprovalProcess.jsp file. Remove NoFilter_ from file name and replace existing file with new emxLifecycleTasksMassApprovalProcess.jsp (That is renamed after removing NoFilter_)

 

WEB.XML

Location: ../WEB-INF/web.xml

Check if following section are available in this file. If available and commented then uncomment these, else add below entries.

 

Note:  Highlighted in red section will already be available in web.xml. You need to add entries in below order only.
