style Tag


Contents

The style tag represents formatting options, such as bold or italic as well as the style of font and the point size of the characters. The style tag is made up by setting attributes on the tag. These are shown in the table below.

 

Name

Value

Notes

name

{Suitable value}

Defines a name for the style, so it can be referenced later.

font

{font name}

Defines the name of the font used in the stamp.  The font must be installed on the server.

size

{Suitable number}

Defines the point size of the font used in the stamp.

format

{bold, italic or underline}

e.g.

bold,underline

Defines the font format used in the stamp. It can hold multiple values separated by a comma and no space.

default

true/false

Sets which style to use as the default style

colour

{Suitable colour}
e.g.

rgb(255,255,255)

Defines the font colour used in the stamp.  Can either be system colour names or an RGB value.

 

Example style XML

  <style name="bold" font="Arial" size="9" format="bold"/>
  <style name="boldUnderline" font="Arial" size="9" format="bold,underline"/>
  <style name="normal" font="Arial" size="9" format="" default="true"/>
  <style name="large" font="Arial" size="14" format="" />
  <style name="H1" font="Arial" size="14" format="bold" />
  <style name="H2" font="Arial" size="12" format="bold" />
  <style name="small" font="Arial" size="5" format="" />
  <style name="smallbold" font="Arial" size="5" format="bold" />
