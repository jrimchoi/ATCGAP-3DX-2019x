headerImage Tag


Contents

The headerImage tag is made up of a several attributes and sub tags whose values define the display of an image in the header of the report.  The image used must be checked in to ENOVIA.

 

headerImage attributes:

scaleToFit

true
false

Scales the image to fit the defined size of the header (set in the margin Tag).

stretch

true
false

Stretches the image to fit the defined size (set in the margin Tag).

overlay

true
false

If set to true, the image will overlay text.

underlay

true
false

if set to true the image will underlay text.

 

headerimage sub tags:

Name

Value

Notes

type

{Suitable type}

The type of the business object that the image is checked into.

name

{Suitable name}

The name of the business object that the image is checked into.

revision

{Suitable revision}

The revision of the business object that the image is checked into.

file-name

{Suitable file name}

The file name of the image that is checked into ENOVIA.

 

Example headerImage XML

<headerimage horizontalPosition="top" verticalPosition="left" scaleToFit="true" stretch="false" underlay="true" >
     <type>Document Template</type>
     <name>PDF-TEMP-0001</name>
     <revision>1</revision>
     <file-name>header.jpg</file-name>
</headerimage>
