document Tag


Contents

The document tag is the root node for the XML configuration an Excel Report Generator Activity, and the following series of sub tags that define the report:

 

Name

Value

Notes

useBlankWorkbook

true

or

false

true means the report will not use a predefined Excel template, and therefore templateType, templateName, templateRevision and templateFileName are not required.

false means the report will use a predefined Excel template.

templateType

{Business object type}

Defines the type of the business object that has the Excel template to be used.

templateName

{Business object name}

Defines the name of the business object that has the Excel template to be used.

templateRevision

{Business object revision}

Defines the revision of the business object that has the Excel template to be used.

templateFileName

{Business object filename}

Defines the filename of the business object that has the Excel template to be used.

queryFilters

{List of queryFilter tags}

Contains a list of queryFilter tags that filter queries

worksheets

{List of worksheet tags}

Contains a list of worksheet tags that define which worksheets in the Excel file data will be exported to.

 

Example document XML

<document>
 <useBlankWorkbook>FALSE</useBlankWorkbook> 
 <templateType>Document Template</templateType> <!-- not needed if useBlankWorkbook set to false -->
 <templateName>drPlantReportExcelTemplate-0001</templateName> <!-- not needed if useBlankWorkbook set to false -->
 <templateRevision>1</templateRevision> <!-- not needed if useBlankWorkbook set to false -->
 <templateFileName>drPlantReportExcelTemplate-0001.xlsx</templateFileName> <!-- not needed if useBlankWorkbook set to true -->
 <queryFilters>
         <queryFilter name="majorComponentFilter" >
                 appliesto bus type Part name * revision * where 'attribute[Component Name] == "${requestParamter:Component}" AND attribute[Component Type] ~~ "${requestParamter:ComponentType}"'
         </queryFilter>
 </queryFilters>
 <workSheets>    
         <workSheet>
         </worksheet>
 </worksheets>
</document>
