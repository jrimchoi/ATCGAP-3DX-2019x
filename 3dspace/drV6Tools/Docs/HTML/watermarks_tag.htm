watermarks Tag


Contents

The watermarks tag holds a list of watermark tags that will generate the stamp for the PDF document.

 

watermarks XML

<watermarks>
   <watermark/>

   <watermark/>

   <watermark/>
   <if/>

   <if/>
   <ifPage/>

   <ifPage/>
</watermarks>

 

 

The watermarks tag can contain the following sub-tags:

Name

Notes

watermark

Watermark elements contain an individual watermark mapping

if

If elements allow you to define conditions. If all the conditions pass then the watermarks the elements contain will be mapped.

ifPage

IfPage element allow you to check the page size. If the page size matches then the watermarks they contain will be mapped.

 

Example watermarks XML

<watermarks>
   <watermark style="large" horizontalPosition="middle" verticalPosition="center" alignment="center" rotation="45" height="200" width="800" underlay="false"> 
        <mql-selectable>current date</mql-selectable> <!-- this is the mql selectable value that gets the value of the attribute that you wished to be stamped -->
   </watermark> 

   <watermark style="H1" horizontalPosition="middle" verticalPosition="bottom" alignment="left" height="25" width="400" underlay="false"> 
        <text>Printed by: </text><value on="user">First Name</value><text> </text><value on="user">Last Name</value> 
   </watermark> 
   <if>
        <condition leftHandSource="mql selectable" leftHandValue="current" condition="=" rightHandSource="fixed value" rightHandValue="Released"/> 
        <elements>
                   <watermark style="large" horizontalPosition="middle" verticalPosition="center" alignment="center" rotation="45" height="200" width="800" underlay="false"> 
               <text>Property Of...</text> <!-- A fixed text value -->
            </watermark> 
        </elements>
   </if>

   <ifPage size="A3" landscape="true">
      <watermark style="H1" horizontalPosition="middle" verticalPosition="top" alignment="left" height="25" width="400" underlay="false"> 
         <text>A3 sheet</text><value on="user">First Name</value>
      </watermark> 
   </ifPage>
</watermarks>
