Copy Values Element Text Manipulation


Contents

The DRLTools Copy Values Element Text Manipulation allows you to pull a basic or attribute value from the object defined in the Source value.

 

To add a Copy Value Element Text Manipulation object:

•Select the Copy Value Elements tab on the Copy Values Target Attribute.
•Click Actions.
•Select Create Copy Value Element Text Manipulation...

oThis will open the DRLTools Copy Value Element Text Manipulation create form.
•Set the following values:

Attribute Name

Value

Type

DRLTools Copy Value Element Text Manipulation

Name

{A suitable identifier}

Revision

{Any suitable value}

Source

This defines the source value. Depending on what is selected in this value drives the value that should be entered into the Value attribute.

Source Options

Description

object

Retrieve the value from the current context object and the Value attribute value can be set any basic or attribute on the object.

user 

Retrieve the value from the current user object and the Value attribute value can be set any basic or attribute on the user object.

relationship 

Retrieve the value from the current context relationship and the Value attribute value can be set any basic or attribute on the relationship.

parent

Retrieve the value from the current context parent object and the Value attribute value can be set any basic or attribute on the parent object.

from 

Retrieve the value from the object on the from end of the current relationship and the Value attribute value can be set any basic or attribute on the object.

to

Retrieve the value from the object on the to end of the current relationship and the Value attribute value can be set any basic or attribute on the object.

file

Retrieve the value from the current file object. NOTE: this value is only usable when 3dxtools is running the Copy Values Activity in the context of a file operation. i.e. Export Target Directory. The Value attribute must be set to either 'File Name', 'Format', 'File Extension' or 'File Name Without Extension'

Value

This value defines the basic or attribute name to be retrieved from the object defined in the Source value. It must be a valid basic or attribute name (for attributes you do not need to include attribute[ ] ) e.g. First Name.

Text Manipulation

Type

This value defines the type of text manipulation that should be applied to the base value. Depending on what value is entered here defines which other values should be set and what they should be set to.

Value

Description

left

Get a set number of characters from the left of the base value. This requires the following values to be set:

Argument 1 - This defines the number of characters from the left to retrieve.

right

Get a set number of characters from the right of the base value. This requires the following values to be set:

Argument 1 - This defines the number of characters from the right to retrieve.

mid

Get a sub string of characters from inside of the base value. This requires the following values to be set:

Argument 1 - This defines the index number for the start character for the copy.

Argument 2 - This defines the index number for the end character for the copy.

split 

Splits the base value into a list of values, using a specific character to split by and then retrieves one value from a specific point in the list. This requires the following values to be set:

Argument 1 - This defines the character to use to split the base value with.

Argument 2 - This defines the index number for the position in the resulting list to retrieve the value from.

trim left 

This removes a set number of characters from the left of the base value. This requires the following values to be set:

Argument 1 - This defines the number of characters to remove from the left.

trim right 

This removes a set number of characters from the right of the base value. This requires the following values to be set:

Argument 1 - This defines the number of characters to remove from the right.

to upper 

This converts the value to all upper case.

to lower 

This converts the value to all lower case.
