Cascade Selection List


Contents

The 3dxtools Cascade Selection List object allows you to create a list control in a Cascade, by which users can pick from a list of values.  Each value in the Cascade Selection List is defined by a Cascade Selection List Option.

 

To add a Cascade Selection List object:

•Select the Cascade Selection tab on the DRLTools Cascade Activity.
•Click Actions.
•Select Create DRLTools Cascade Selection List...

 

oThis will open the DRLTools Cascade Selection List create form.
•Set the following values:

True will check for any If Condition objects linked to the Selection List object, to see if the Selection List control should be displayed or not.  The Selection List control will be displayed if all the If Condition objects evaluate to true.

False does will not check for any If Condition objects.
