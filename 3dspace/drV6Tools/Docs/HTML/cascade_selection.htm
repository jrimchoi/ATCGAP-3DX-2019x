Cascade Selection objects


Contents

The Cascade Selection objects define the controls that are displayed in a Cascade, allowing the administrators to design the Cascade front end.

The example below shows a Cascade Selection Free Text object, which defines a Cascade Selection Free Text control.

 

An example of Cascade Selection Free Text control is shown below.

 

To add Cascade Selection objects follow the steps below:

•Select the Cascade Selection tab on the DRLTools Cascade Activity.
•Click Actions
•Select one of the Create Cascade Selection Object... commands.

 

Each Cascade Selection object defines a different type of control, allowing administrators to design a versatile front end for the Cascade.

Cascade Selection Object

Description

Cascade Selection Date

Adds a calendar control.

Cascade Selection Free Text

Adds a text box control.

Cascade Selection Group

Adds a group of controls.

Cascade Selection List

Adds a list of selectable items.

Cascade Selection List Option

Represents a list item in a Selection list control.

Cascade Selection Search List

Allows to create a search query defined in the configurations and the results that match the query are returned as a list.

Cascade Selection Simple List

Represents a simple list of fixed values defined in the configuration and displayed as selectable items.
