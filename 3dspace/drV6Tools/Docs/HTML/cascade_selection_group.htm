Cascade Selection Group


Contents

The 3dxtools Cascade Selection Group object allows you to define a container control that is used to place other related controls in a group.

 

To add a Cascade Selection Group object:

•Select the Cascade Selection tab on the DRLTools Cascade Activity.
•Click Actions.
•Select Create DRLTools Cascade Selection Group...

oThis will open the DRLTools Cascade Selection Group create form.
•Set the following values:
