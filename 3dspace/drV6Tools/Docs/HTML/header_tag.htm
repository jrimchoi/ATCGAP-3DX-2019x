header Tag


Contents

The header tag holds all the information for the header of the page. This can include an image (defined in a headerImage tag) and text values (defined in headerText tags).

 

Example header XML

<header>
 <headerimage horizontalPosition="top" verticalPosition="left" scaleToFit="true" stretch="false" underlay="true" >
         <type>Document Template</type>
         <name>PDF-TEMP-0001</name>
         <revision>1</revision>
         <file-name>header.jpg</file-name>
 </headerimage> 
 <headertext style="H1" horizontalPosition="right" verticalPosition="top" alignment="right" height="80" width="600">
         <newline/>
         <text>EQUIPMENT DATA SHEET</text>
 </headertext>
 <headertext style="H2" horizontalPosition="left" verticalPosition="bottom" alignment="left" height="70" width="600">
         <value>Structured Data Sheet Template</value>
 </headertext>
 <headertext style="H1" horizontalPosition="left" verticalPosition="bottom" alignment="left" height="50" width="300">
         <value>Description</value>
 </headertext>    
 <headertext style="bold" horizontalPosition="right" verticalPosition="bottom" alignment="left" height="70" width="150">
         <text>Document No: </text><value>name</value>
 </headertext>
 <headertext style="bold" horizontalPosition="right" verticalPosition="bottom" alignment="left" height="50" width="150">
         <text>Date: </text><format formatString="dd-MMM-yyyy" type="date"><value>Originated</value></format>
 </headertext>
 <headertext style="bold" horizontalPosition="right" verticalPosition="bottom" alignment="left" height="30" width="150">
         <text>Page: &lt;%PageNo%&gt; of &lt;%PageCount%&gt;</text>
 </headertext>
</header>
