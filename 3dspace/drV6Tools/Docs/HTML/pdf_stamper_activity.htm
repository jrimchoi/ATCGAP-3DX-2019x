PDF Stamper Activity


Contents

The 3dxtools PDF Stamper Activity allows PDF documents to be stamped or watermarked. It is configured with XML.

Configuration and Settings

To setup a 3dxtools PDF Stamper Activity:

1. Add an PDF Stamper Activity➣

•Select the Dependant Activities tab on either a top level 3dxtools Triggers action object or another 3dxtools Activities object.
•Click Actions.
•Select Create DRLTools PDF Stamper Activity...

3. Configure the XML layout for the Excel Report Generator Activity ➣

The Report Layout attribute on the PDF Stamper Activity contains the full definition of the PDF Stamp in an XML configuration.

The XML starts with a root document tag that contains all the other tags required.  Within the document tag are styles (defining the look of the stamp) and pages (defining the placement of the stamp). Styles holds a list of style tags that can be used across the document and pages holds a list of page tags. A page has name, pageRange, unitOfMeasure and watermarks sub tags. Watermarks contains watermark sub tags that have several attributes and hold the values which will be stamped on the PDF.

 

Full list of PDF Stamper Report Layout tags ➣

Tag Name

Notes

document

The root tag, contains all other tags.

styles

Contains a list of style tags.

style

Defines a single style.

pages

Contains a list of page tags.

page

Contains name, pageRange, unitOfMeasure and watermarks tags.

name

Defines the name of a page.

pageRange

Defines the number of pages to be stamped.

unitOfMeasure

Defines the units that the page uses.

watermarks

Contains a list of watermark tags.

watermark

Defines a watermark value.

 

Full Report Layout XML Example ➣

<document> 
 <styles> <!-- These are font styles, sizes and formats that can be added and deleted as you wish to use them. The name is a reference to that particular style that can be used later on  -->
         <style name="bold" font="Arial" size="9" format="bold"/> 
         <style name="normal" font="Arial" size="9" format="" default="true"/> 
         <style name="large" font="Times New Roman" size="100" format="bold,underline" colour="red" strokeColour="blue" strokeWidth="10" /> 
         <style name="H1" font="Arial" size="14" format="bold" colour="red" /> 
         <style name="H2" font="Arial" size="10" format="bold" colour="green"/> 
         <style name="small" font="Arial" size="5" format="" /> 
         <style name="smallbold" font="Arial" size="5" format="bold" /> 
 </styles> 
 <pages> <!-- These are the pages that are to be stamped  -->
         <page> <!-- Individual Page to be stamped  -->
                 <name>FirstPage</name> <!-- name of the page for reference  -->
                 <pageRange>1</pageRange> <!-- range of pages to be stamped (- for all)  -->
                 <unitOfMeasure>MM</unitOfMeasure> <!-- The unit of measure that all the height and width tags will use  -->
                 <watermarks> <!-- The water marks or what will actually be stamped -->
                         <!-- These are the configurations for the watermark and control where abouts on the page they get aligned  -->
                         <watermark style="large" horizontalPosition="middle" verticalPosition="center" alignment="center" rotation="45" height="200" width="800" underlay="false"> 
                                 <mql-selectable>current date</mql-selectable> <!-- this is the mql selectable value that gets the value of the attribute that you wished to be stamped -->
                         </watermark> 
                 </watermarks> 
         </page>
 </pages> 
</document>
