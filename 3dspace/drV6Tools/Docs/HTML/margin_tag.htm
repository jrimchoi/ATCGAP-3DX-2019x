margin Tag


Contents

The margin tag holds all configuration information for what size margins the PDF document uses. This tells the report generator how much white space is on each edge of the page before it starts writing text. All the values are given in the predefined unit of measure.

 

margin attributes:

Name

Value

Notes

top

{suitable number}

Top margin size.

left

{suitable number}

Left margin size.

right

{suitable number}

Right margin size.

bottom

{suitable number}

Bottom margin size.

headerHeight

{suitable number}

Height of the header.

footerHeight

{suitable number}

Height of the footer.

 

Example:

<margin>
 <top>5</top>
 <left>20</left>
 <right>20</right>
 <bottom>10</bottom>
 <headerHeight>55</headerHeight>
 <footerHeight>20</footerHeight>

</margin>
