subRepeaters Tag


Contents

These are rowRepeaters that are used inside a rowRepeater tag. They hold more rowRepeater tags and are used for expand statements on a row that has already been fetched by a query.

 

Example subRepeaters XML

<subRepeaters> 
  <rowRepeater> 
  <startRow>0</startRow> <!--This start row is the row to start counting from in relation to what row the repeater is currently at-->
  <query queryType="expand"  objectidposition="6" relationshipidposition="7"  orderby="id" dumpdelimiter="|" typePattern="Part" wherePattern='' retrieveAllBasics="true" attributes="Component Manufacturer,Component Name,Component Type" relationshipPattern="Major Component" relationshipAttributes="Component Certificate Number,Engine Stage,Engine Tier,Date in Service,Relationship State" filter="majorComponentFilter"></query>
  <cells> 
    <cell column="O"> 
      <value>Component Manufacturer</value>
    </cell>
    <cell column="P"> 
      <value>Component Name</value>
    </cell>
    <cell column="Q">
      <value>Component Type</value> 
    </cell>
    <cell column="R"> 
      <value on="relationship">Component Serial Number</value> 
    </cell>
    <cell column="S">
      <value on="relationship">Component Certificate Number</value>
    </cell>
    <cell column="T">
      <value on="relationship">Engine Stage</value>
    </cell>
    <cell column="U">
      <value on="relationship">Engine Tier</value>
    </cell>
    <cell column="V">
      <value on="relationship">Date in Service</value>
    </cell>
    <cell column="W">
      <value on="relationship">Relationship State</value>
    </cell>
  </cells>      
  </rowRepeater>
</subRepeaters>
