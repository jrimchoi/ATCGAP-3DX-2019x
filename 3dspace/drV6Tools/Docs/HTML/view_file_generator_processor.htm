View File Generator Processor


Contents

The 3dxtools View File Generator Processor is designed to generate PDF view files for objects checked into the ENOVIA store. As part of the process the system will open the file in its native application, carry out parameter mappings (i.e. signoff signatures) to the document and create a PDF.

 

The currently support file formats are:

o Microsoft Word
o Microsoft Excel
o Microsoft PowerPoint
o AutoCAD Drawings
o CATIA V5 Drawings
o SolidWorks Drawings
o Inventor Drawings

 

NOTE: Although the 3dxprocessor Job Server can run as a service some of the supported file formats (in particular SolidWorks Drawings) require a logged in windows user session to execute successfully.

 

Configuration and Settings

1. Add a drV6Tools Job Server View File Generatorr object ➣

The main configuration for the View File Generator is controlled using a Job Server View File Generator object. Connected to this are View File Generator Settings objects that hold the specific settings for each of the file types that can be processed. To add Job Server View File Generator object:

•Select the Processors tab on the Job Server Controller
•Click Actions.
•Select Create Job Server Processor...
•Select the Type Selector
•Select the Type Processor View File Generator

2. Add the required View File Processor Settings objects ➣

Each of the file types (CATIA Drawings, SolidWorks Drawings, etc) have their own configuration requirements to allow them to be processed. To accommodate this each file type has its own View File Processor Settings object that contains those specific settings. These are created and connected to the main View File Processor object. Any or all of the specific View File Generator Settings object can be associated with a View FIle Processor object, the system will use the file extension of the file to be processed to select the particular one to use at run-time.

To add Job Server View File Generator Settings object:

•Select the VIew File Generator Settings tab on the View File Generator Processor object.
•Click Actions.
•Select Create Processor View File Generator Base Setting...
•Where applicable select the Type Selector
•Select the Type for the file extension you wish to process. the table below list to available options.
