document Tag


Contents

The document tag holds all the information for the XML configuration of a PDF Stamper Activity, and is the root node for the report generator.  A document holds Styles and Pages, which make up the core of the stamp.

 

Example document XML

<document>
   <styles>
      <style/>
      <style/>
      <style/>
      <style/>
   </styles>
   <pages>
      <page/>
      <page/>
   </pages>
</document>
