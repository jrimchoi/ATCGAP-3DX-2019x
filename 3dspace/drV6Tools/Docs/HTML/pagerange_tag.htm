pageRange tag


Contents

The pageRange tag defines the number of pages the page tag will stamp. If you wish to stamp all pages set the value to a hyphen ( - ).

 

Example pageRange XML

This example will stamp the first page:
<pageRange>1</pageRange> <!--Will stamp the first page-->

 
This example will stamp all pages:
<pageRange>-</pageRange> <!--Will stamp all pages-->
