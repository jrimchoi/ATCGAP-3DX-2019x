Copy Values Element Referenced Object Attribute


Contents

The DRLTools Copy Values Element Referenced Object Attribute allows you to pull a basic or attribute value from a referenced object. The intended use for this Copy Value element is when an attribute on the object contains the name value of another object, and you wish to copy a value from that named object. It uses a fixed ENOVIA Type and Revision (defined in Referenced Object Type and Referenced Object Revision) and the Name value is calculated using a combination of Source and Value. The value to retrieve from the referenced object and use as the result of this Copy Value element is defined in Referenced Object Attribute.

 

To add a Copy Value Element Referenced Object Attribute:

•Select the Copy Value Elements tab on the Copy Values Target Attribute.
•Click Actions.
•Select Create Copy Value Element Referenced Object Attribute...

oThis will open the DRLTools Copy Value Element Referenced Object Attribute create form.
•Set the following values:

Source

This value defines on which object the attribute value defined in Value should be retrieved from. This value is then used as the ENOVIA business object Name  to use to search for the reference object. The list of possible options are defined below.

Source Options

Description

object

Retrieve the value from the current context object.

user 

Retrieve the value from the current user object.

relationship 

Retrieve the value from the current context relationship.

parent

Retrieve the value from the current context parent object.

from 

Retrieve the value from the object on the from end of the current relationship.

to

Retrieve the value from the object on the to end of the current relationship..

Value

{Valid basic or attribute name} 

Defines the basic or attribute name to be used to retrieve the objects name value from the object defined in the Source.
