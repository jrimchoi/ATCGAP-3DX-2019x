mql-Selectable Tag


Contents

The mql-selectable tag runs the select command on a business object that is fetched by the query inside a rowRepeater.  This tag has one attribute called on. The values on accepts are shown in the table below. If on is left blank it will default to object.

Each attribute defined in a mql-selectable tag should also be present in the query tag mqlSelectables attribute to increase performance.

Value

Notes

object

The value is fetched from the currentObject

from

The value is fetched from the fromObject

to

The value is fetched from the toObject

parent

The value is fetched from the parentObject

relationship

The value is fetched from the relationshipObject

user

The value is fetched from the current userObject

useradmin

The value is fetched from the current adminUserObject

 

Example mql-selectable XML

<cell column="R"> 
   <mql-selectable>relationship[Part to Plant Number].from.relationship[TLPN].from.attribute[Item Number Rating]</mql-selectable>
</cell>
