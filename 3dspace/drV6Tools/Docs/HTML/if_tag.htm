if Tag


Contents

 

An if tag holds a condition tag which when equal to true will populate the cells contained within the elements tag. The if tag condition is made up of five attributes which tell the condition where to find the left and right hand sides of the equation and the condition on which to evaluate to true.

The condition tag builds up an equation that looks like the following:

if leftHandValue(from leftHandSource) condition(<,>,=,!=...) rightHandValue(from rightHandSource) then...

 

 

The attributes for a condition tag are:

Name

Value

Notes

leftHandSource

{see table below}

The relevant value for the chosen source

leftHandValue

{Suitable value}

condition

{<,>,=,!=,regex match,

regex not match,contains,

does not contain}

if set to regex match or regex not match then Right Hand Source should be set to expression and Right Hand Value should be set to a valid regular expression statement.

rightHandSource

{see table below}

The relevant value for the chosen source.

rightHandValue

{Suitable value}

 

The table below shows the available options for the leftHandSource and rightHandSource:

Value

Notes

attribute

Fetches the value from a named attribute

fixed value

The value entered is used

expression

Evaluates the expression entered

mql selectable

Fetches the value from an MQL expression

mql command

Fetches the value from an MQL command

user business object

Fetches a user business object

user business object mql selectable

Fetches an MQL selectable value from a user business object

user admin object

Fetches a named user admin object

lookup

Fetches the value from a lookup object

 

Example if XML

<if>
 <condition leftHandSource="attribute" leftHandValue="Structured Data Sheet Number2 Alt" condition="=" rightHandSource="fixed value" rightHandValue="" />
         <elements>     
                 <cell rowspan="1" columnspan="2" evenbackgroundcolour="rgb(220,220,220)">
                         <value>Structured Data Sheet Boolean</value>
                         <value>Structured Data Sheet Text2</value>
                         <value>Structured Data Sheet Number2</value>
                 </cell>
         </elements>        
</if>
