1.Create Action commands in ENOVIA
✍Setup Actions commands for Reports

 

•Create 3dxtoolsCreateWorkspaceDocumentReport as below:

Href: ${ROOT_DIR}/3dxtools/3dxtools.jsp?drtoolsKey=WorkspaceDocumentReport

Settings – Image: ${COMMON_DIR}/images/iconActionCreate.gif

 

•Create 3dxtoolsCreateBatchWorkspaceDocumentReport as below:

Href: ${ROOT_DIR}/3dxtools/drCascade.action?drtoolsKey=BatchWorkspaceDocumentReportCascade&action=init&mode=Forward

Settings – Image: ${COMMON_DIR}/images/iconActionCreate.gif

Label: Create Batch Workspace Contents Report...

•Create 3dxtoolsContentSummaryCreateNew menu as below:

 

Add 3dxtoolsCreateWorkspaceDocumentReport and 3dxtoolsCreateBatchWorkspaceDocumentReport commands in this menu

Add this menu in OOTB TMCContentSummaryTopActionBar menu.

 

•Create drlCreateIssuesStatusReport command as below:

Href: ${ROOT_DIR}/3dxtools/3dxtools.jsp?operation=RunDRToolsServerCommandActionTrigger&amp;drToolsKey=Create_Issues_Status_Report

Settings – Image: ${COMMON_DIR}/images/iconActionCreate.gif

Label: Issues Status Report

 

 

•Create 3dxtoolsDemoMyDesk menu as below:

 

Add drlCreateIssuesStatusReport command in this menu

Add this menu in OOTB My Desk menu.

 

•Create 3dxtoolsCreateDRLDocumentDataPDFReport command as below:

Href: ${ROOT_DIR}/3dxtools/3dxtools.jsp?drtoolsKey=PDFReportGeneratorTypeActionTrigger

Settings – Image: ${COMMON_DIR}/images/iconActionCreate.gif

Label: Create PDF Report

 

•Create 3dxtoolsTMCDocumentsActionToolbarMenu menu as below:

5.Create Cascade Activity object
✍To create a DRLTools Cascade Activity

 

•Open BatchWorkspaceDocumentReportCascade DRLTools Type Action Trigger and select the Dependant Activities tab
•Click Actions
•Select Create DRLTools Cascade Activity

 

•Fill the following details in Create DRLTools Cascade Activity form
•The fields are explained below.

 

 

Attribute Name

Value

Notes

Type

DRLTools Cascade Activity

This should already be set for you.

Name

BatchWorkspaceDocumentReportCascade 

Any suitable value

Description

Batch Workspace Document Report Cascade

Any suitable value

Relationship is From Source Object

TRUE

 

Applicable if Mode is Add. This value defines which of the objects at either end of the relationship should be set as the current context object.

If TRUE the relationship is from the source object.

Mode

Forward

Forward means the Cascade Activity will forward the selections to the URL defined in Target URL

Target URL

3dxtools.jsp?drToolsKey=BatchWorkspaceDocumentReportQueueObject&

Applicable if Mode is set to Forward. {URL} is the URL that the Cascade Activity will forward to

Values To Append To Target URL

NameFilter

Applicable if Mode is set to Forward. {URL parameters} are the parameters to be appended to the URL defined in Target URL (and will usually be names of form controls).  Multiple parameters must be comma-separated.

Header

Workspace Report (server side)

Header is the header text displayed to the user at the top of the Cascade.

Sub Header

Report generated by processor system

Sub Header is the sub header text displayed to the user underneath the Header Text.
•Click on Done.

DRLTools Cascade Activity is created successfully.
