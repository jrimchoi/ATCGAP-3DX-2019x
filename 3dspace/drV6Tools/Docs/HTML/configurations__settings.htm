9.Create New From Template Activity object

To create a DRLTools New From Template Activity:

•Open Create_DRL DOCUMENTS Lookup Values DRLTools Lookup Values Activity and select the Dependant Activities tab
•Click Actions
•Select Create DRLTools New From Template Activity.

•Fill the following details in Create New From Template Activity form
•The fields are explained below.

Name

Value

Notes

Type

DRLTools New From Template Activity

This should already be set for you

Name

DRL DOCUMENTS Set Template 

Any suitable value

Description

DRL DOCUMENTS Set Template 

Any suitable value

Check For Files

TRUE

This defines if the system should first check to see if any files are already being managed by the business object before adding the new file from the template.

Check For Files With Extensions

doc,docx,docm,xls,xlsx,xlsm

{List of file extensions}

If Check For Files is set to TRUE then this value can be used to check the file extension of any existing files on the current context object. If any match the file extensions defined here, then the system will not add the template file. The file extensions should be entered without the preceding “.”. Multiple file extensions should be comma delimited with no following space.

 

File Name

${name}-${Title}-v${revision}

File name to use for copied document template.

Filter

${lookup:Doc Template}

Filter is used to identify document template based on Doc Template column in lookup reference file.

 

•Click on Done.

New From Template Activity is created successfully.

•Update Document Templates.xml file to add/modify the templates

•Add checkin command in checkin_templates.mql file

..\3dxtools.TestPlatform\Schema\businessobject\templates\checkin_templates.mql

 

•Put files in templates directory

..\3dxtools.TestPlatform\Schema\businessobject\templates

 

•When you install the package, Document Templates will be created with corresponding reference file from templates directory.

•Select New From Template Activity Templates tab in properties page

•Click Actions.
•Select connect.

It will open a search page. Search for Document Template and select below document templates and click connect.

 

•New From Template Activity Templates are connected successfully.
