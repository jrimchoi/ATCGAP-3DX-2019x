if Tag


Contents

The if tag allows you to define one of more condition objects and elements to run if all the conditions pass.

if XML

<if>
   <condition/>

   <condition/>

   <condition/>
   <elements/>
</watermarks>

 

 

if condition XML

The condition tag allows you to define a single condition. You can set multiple condition objects in an if tag and all must pass to allow it to run the sub elements.

The configuration value options for the if condition tag are exactly the same as the If Condition Object

 

<condition leftHandSource="" leftHandValue="" condition="" rightHandSource="" rightHandValue=""/> 

 

Condition Element Attributes:

Left Hand Source

This defines the source value for the left hand side of the comparison and can be set to one of the following values. Depending on what is selected in this value drives the value that should be entered into the Left Hand Value attribute. (see the If Condition Object for more details)

Left Hand Value

This value defines the left hand value to be used, based on and valid for the Left Hand Source value that has been selected (see the If Condition Object for more details).

Condition

The condition type to use to compare the left and right hand values with (see the If Condition Object for more details).                

Right Hand Source

This defines the source value for the right hand side of the comparison and can be set to one of the following values. Depending on what is selected in this value drives the value that should be entered into the Right Hand Value attribute (see the If Condition Object for more details).

Right Hand Value

This value defines the right hand value to be used based on and valid for the Right Hand Source value that has been selected (see the If Condition Object for more details).

 

Example if condition XML

<if>
   <condition leftHandSource="mql selectable" leftHandValue="current" condition="=" rightHandSource="fixed value" rightHandValue="Released"/> 
   <condition leftHandSource="mql selectable" leftHandValue="revision" condition="!=" rightHandSource="mql selectable" rightHandValue="last"/> 
   <elements>
      <watermark style="large" horizontalPosition="middle" verticalPosition="center" alignment="center" rotation="45" height="200" width="800" underlay="false"> 
         <text>Property Of...</text> <!-- A fixed text value -->
      </watermark> 
   </elements>
</if>
