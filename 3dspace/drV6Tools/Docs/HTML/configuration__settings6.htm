To setup the 3dxtools Configurable Document Actions Commands:

1. Find OOTB ENOVIA Table Admin Objects ➣

To enable the 3dxtools Configurable Show Process Status on column of the table,

•In ENOVIA MQL find the ENOVIA Table you wish to use.
•Go to Windows Start Menu ->mql ->click on "Start Mql.exe".
•set the context by writing below query,

  MQL>set context user creator;

•find the table you wish to use by querying like  below,

  MQL>list type ${table_name};

•Add a new Column "Process Status" to table TMCContentSummary

Syntax:

MQL>mod table ${table_name} system add column ${column_name} order ${order_number};

Example:

MQL>mod table TMCContentSummary system add column "Process Status" order 6;

•Add below settings on the table column,

Setting Name

Value

Notes

program

drV6ToolsJPO

This  points the system at the 3dxtools version of the JPO.

method

getIconOndrlStatusUpdate

This is the method in the JPO to run

Column Type

programHTMLOutput

This defines the type Column.

dr show status for processors

{name of the Processor ,multiple processor name can be given with comma separator.}

This defines the name of the Processor which will be used as where condition to get queue objects ,Eg: if the Processor is ViewFileProcessor ,then get the status of queue object, by default it can be empty.

dr show queue object status comments as tooltip

{true or false}

This defines the tooltip that will be displayed if the users hovers the mouse over the command icon , it will give value of  "Processor Comments " if the queue object is failed.

dr icon link to queue object on failure

{true or false}

This defines the hyperlink to queue object for admin user if the Queue object is failed to process.

dr icon link to queue object for admin

{true or false}

This decides whether hyperlink should be given to Process Failed Objects for administrator user or not.

dr show processed icon

{true or false}

This decides whether icon should be dispalyed fro Processed objects or not.

Example:To add settings to column of the table,

MQL>mod table TMCContentSummary system field mod name "Process Status"  setting "dr show status for processors" "ViewFileProcessor" settting "dr show queue object status comments as tooltip" true;

2. Find  OOTB ENOVIA Web Form Admin Objects ➣    

To enable the 3dxtools Configurable Show Process Status on field of the form,

•find the table you wish to use by querying like  below,

  MQL>list type ${form_name};

•Add a new Column "Process Status" to form drV6ToolsDRLDOCUMENTSProperties

Syntax:

MQL>mod form ${form_name} field ${field_name} order ${field_number};

Example:

MQL>mod form drV6ToolsDRLDOCUMENTSProperties field "Process Status" order 6;

•Add below settings on the web form field,
