Management Processor


Contents

The 3dxprocessor Job Server Management  Processor is designed  to help manage the Queue Objects and the administration of the 3dxprocessor. This plugin will enable functions such are : purging of Processed queue objects after a certain amount of time; moving the processing details output out of the database field and into a text file (to save on database space): sending daily summary email regarding Processed and Failed Queue objects.
