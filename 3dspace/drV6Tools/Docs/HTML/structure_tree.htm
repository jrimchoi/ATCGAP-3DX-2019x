Structure Tree


Contents

Structure Trees are a tree menu that allows a user to navigate through related items without having to launch a new window each time a link is clicked. It also tracks nested relationships in a tree structure.

6. Create Configuration Setting Object in Matrix➣

If the structure menu is required to only show specific relationships then the a DRLTools General Configuration Object will need to be created.

The name of this object must match the name of the type that has the required relationships on with StructureTree_ appended to the start of it. For example if a DRLTools General Configuration Object was created to manage the DRLTools Folder type, then the name would be StructureTree_DRLTools Folder. Only 2 values are used. drlArg1 and drlArg2.

Attribute Name

Value

Notes

drlArg1

Suitable text value

Will add any from relationship that contains this text value. For example:

If set to DRLTools any from relationship that contains DRLTools will be added

drlArg2

Suitable relationship names separated by |

This will add specific relationships separated by a pipe (|)
