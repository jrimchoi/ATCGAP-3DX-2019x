Cascade Selection Search List


Contents

The 3dxtools Cascade Selection Search List object allows you to define a list based on an MQL query.  The query can use variables input by the user.

 

To add a Cascade Selection Search List object:

•Select the Cascade Selection tab on the DRLTools Cascade Activity.
•Click Actions.
•Select Create DRLTools Cascade Selection Search List...

 

oThis will open the DRLTools Cascade Selection Search List create form.
•Set the following values:

Attribute Name

Value

Notes

Type

DRLTools Cascade Selection Search List

This should already be set for you.

Name

{A suitable identifier}

{A suitable description}

Check Conditions

True

or

False

True will check for any If Condition objects linked to the Cascade Selection Search List object, to see if the Selection Search List control should be displayed or not.  The Selection Search List control will be displayed if all the If Condition objects evaluate to true.

False does will not check for any If Condition objects.

Enable In Modes

All

or

Add

The default value is All, which includes Add.

Not in use yet.

Not in use yet.

or

False

True displays a header on the Cascade Selection Search List control.

False does not display a header on the Cascade Selection Search List control.

Whether the control should be placed horizontally or vertically.

Display Separator

Defines the separator to use in the displayed search results.

Display Attributes Indices

{List of Attribute Indices}

Specify the list of attributes indices to be displayed. Separate multiple indices with a pipe ( | ).

{MQL search query}

Defines the MQL search query.

MQL Dump Delimiter

{Suitable delimiter}

Specifies the delimiter to be used by the MQL search result dump.

{Index number}

Specifies the MQL result index number.

True

or

False

True means the result index attribute is an Object ID.

False means the result index attribute is not an Object ID.
