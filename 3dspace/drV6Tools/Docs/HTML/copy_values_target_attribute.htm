Copy Values Target Attribute


Contents

Each value to be set by the Copy Values Activity is defined using a 3dxtools Copy Values Target Attribute object. Each one tells the system the target object to set the value on (i.e. the current object, the parent object, the current relationship, etc) and the name of the attribute to set. It is possible to control the order these values are set using the Order Number value on the relationship between the Target Attribute and the Copy Values Element.

 

To configure a Copy Values Target Attribute object:

•Select the Target Attributes tab on the Copy Values Activity object.
•Click Actions.
•Select Create DRLTools Copy Values Target Attribute...

{Attribute name} is the attribute name you want to set on the object defined in the Target Object value.

Update Version Object

{TRUE or FALSE}

If the Target Object is set to object and this value is set to TRUE then the system will also check for and update the same attribute on any linked latest and active version objects.
•Define the Copy Value Elements to generate the value itself.

The Copy Values Element object provides the ability to calculate a single value. Several of these objects can be linked to a Copy Values Target Attribute to construct more complex values, using the Order Number attribute on the relationship to control the order in which the values are calculated and added together. For instance if you wanted to generated a users display name you could create three elements: the first to get the users First Name, the second to add a space and the third to get the users Last Name. All three values get concatenated into a single value that is then applied to the Target Attribute defined in the Copy Values Target Attribute. To add a Copy Value Elements object:

•Select the Copy Value Elements tab on the Copy Values Target Attribute object.
•Click Actions.
•Select one of the Create Copy Value Element... commands.

Each of the Copy Value Element sub types provide different functionality to allow you to calculate values in different ways and are defined below:

Description

Perform text manipulations like left, mid, split on values from one of the current context business objects or relationship

Perform date time calculations and formatting from either the current date/time or one from a value on an object.
