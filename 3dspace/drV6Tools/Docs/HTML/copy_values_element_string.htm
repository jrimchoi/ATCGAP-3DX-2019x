Copy Values Element String


Contents

The DRLTools Copy Values Element String allows you to define a fixed string value to be copied.

 

To add a Copy Value Element String object:

•Select the Copy Value Elements tab on the Copy Values Target Attribute.
•Click Actions.
•Select the Create Copy Value Element String... command

•This will open the DRLTools Copy Value Element String create form.
•Set the following values

Attribute Name

Value

Notes

Type

DRLTools Copy Value Element String

This should already be set for you

Name

{A suitable identifier}

Revision

{Any suitable value}

Value

{Fixed string text}

Defines a fixed string value to be copied.

{Any Macro Value}

Defines a Macro Value that will be parsed by Copy Values.

{SPACE}

{SPACE} (including the curly braces) will be changed to a single space character. This is required since the ENOVIA web UI trims all white spaces from the value before saving it to the database.
