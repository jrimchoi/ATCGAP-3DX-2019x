unitOfMeasure Tag


Contents

The unitOfMeasure tag defines what unit all size measurements are given in, such as millimeters (mm).

Value

MM

INCHES

PIXELS

POINTS

TWIPS

Example unitOfMeasure XML
<unitOfMeasure>MM</unitOfMeasure>
