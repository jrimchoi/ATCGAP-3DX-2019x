rowIf Tag


Contents

The rowIf tag specifies a specific condition that determines if the rowRepeater tag will populate the current row with data. It is made up of a leftHandSource, leftHandValue, condition, rightHandSource and rightHandValue tag. These represent the following structure:

 

if leftHandValue(from leftHandSource) condition(<,>,=,!=) rightHandValue(from rightHandSource) then...

 

Example rowIf XML

This example evaluates the result of an MQL statement and checks that it is greater than 0.

<rowIf>
 <leftHandSource>mql statement</leftHandSource> 
 <leftHandValue>eval expression 'count TRUE' on expand bus ${ID} from relationship "Major Component" type "Part" recurse to 1 filter majorComponentFilter</leftHandValue>
 <condition>greater than</condition>
 <rightHandSource>fixedvalue</rightHandSource>
 <rightHandValue>0</rightHandValue>
</rowIf>

 

 

The leftHandValue tag is the value that the rowIf will use to populate the left hand side of the if condition with, using the leftHandSource value to instruct it where to fetch the value from.

 

Example leftHandValue XML

This example will compare a value returned from an MQL command.

<leftHandSource>mql command</leftHandSource> 
<leftHandValue>eval expression 'count TRUE' on expand bus ${ID} from relationship "Major Component" type "Part" recurse to 1 filter majorComponentFilter</leftHandValue>

 

 

The leftHandSource tag selects where the left hand side of the if condition fetches the value from. It can hold one of the following values:

 

Value

Notes

attribute

Fetches the value from a named attribute

fixed value

The value entered is used

expression

Evaluates the expression entered

mql selectable

Fetches the value from an MQL expression

mql command

Fetches the value from an MQL command

mql statement

Fetches the value from an MQL statement

user business object

Fetches a user business object

user business object mql selectable

Fetches an MQL selectable value from a user business object

user admin object

Fetches a user admin object

lookup

Fetches the value from a lookup object

 

Example leftHandSource XML

<leftHandSource>mql statement</leftHandSource> 

 

 

The condition tag for the if condition to compare two expressions. It can hold one of the following values:

 

Value

greater than

less than

equal

not equal

 

Example condition XML

<condition>greater than</condition>

 

 

The rightHandValue tag is the value that the rowIf will use to populate the right hand side of the if condition with, using the rightHandSource value to instruct it where to fetch the value from.

 

Example rightHandValue XML

This example will compare a fixed value of 0.

<rightHandSource>fixedvalue</rightHandSource>
<rightHandValue>0</rightHandValue>
