Derived Output Symbolic Name

{Symbolic name of the type Derided Output}

it will be "type_DerivedOutput"

Derived Output Relationship Name

{Relationship Name of derived output}

It will be "Derived Output"

Derived Output Policy Symbolic Name

{Policy Name which governs Derived Output} 

It will be "policy_DerivedOutputPolicy"

Derived Output Object Generator Name

{Name of Object Generator for Derived Output}

It will be "type_DerivedOutput"

{Path of an image file}
