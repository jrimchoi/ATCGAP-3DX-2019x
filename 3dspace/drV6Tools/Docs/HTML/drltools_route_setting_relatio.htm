DRLTOOLS Route Setting Relationship


Contents

 This relationship will be used to connect the OOTB route objects/relationships with the DRLTools Route Setting objects. Using this relationship, it should be possible to connect a Route Template object or a Route Node relationship to a DRLTools Route Setting object. The relationship will contain two attributes that will let you define the Response and the Timing for the operations.

 

 drlRouteResponse – This attribute contains a string value. This is used to define the response on which the route settings to be performed. It contains the following range of values.  Start, Signature Reset, Reject, None, Ignore, Approve, Abstain, Any

 

 drlRouteActionTiming – This attribute contains String value with a range of “Before” and “After”. This is used to define the actions to be performed before or after the route or route task action.
