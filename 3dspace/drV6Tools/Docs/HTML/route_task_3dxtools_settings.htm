Route Task 3dxtools Settings


Contents

 It will provide access to the DRLTools Route Setting connected to the Route Task using the DRLTools Route Setting Relationship. This will allow user to define the actions that should be performed at the actions performed on Route Tasks. The Response values can be Start, Signature Reset, Reject, None, Ignore, Approve, Abstain, Any and Timing can be either Before or After. If set to Before that the actions will run before the Route is Started/Ended and if after it will run after the operation has completed.

 

This channel is accessible from two navigations.

 

1)Route Template à 3dxtools Settings à Task Settings.

 

 

Following Actions commands are available in this view:

 

 

 

 

User can perform create, connect, disconnect, delete and export settings operations in this view.

 

Create DRLTools Route Setting: This is a single select command and applicable for task object selection. Newly created Route Setting object will be connected to the route node relationship between selected task and route template.

 

Connect: This is a multi-select command to add existing route setting objects. User can select multiple tasks and search the route setting objects, object(s) selected in search window will be connected to the respective route node relationships of all selected tasks.

 

This view also provides the expanded structure for the task setting configuration and allows user to configure the Response and Timing attributes for the tasks settings.
