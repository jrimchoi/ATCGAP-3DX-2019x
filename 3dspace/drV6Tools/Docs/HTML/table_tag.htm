table Tag


Contents

The table tag holds all the information for a table. A table contains user defined values and values dynamically created from a predefined query. A table is the core of a report as it is how the generator populates the report with data. A table is made up of a series of attributes and several subtags.

 

table attributes:

Name

Value

Notes

x

{numerical value}

x coordinate value on the page for where to start drawing the table

y

{numerical value}

y coordinate value on the page for where to start drawing the table

columns

{numerical value}

Number of columns

columnWidth

{numerical value}

Width of columns

cellHeight

{numerical value}

Height of cells

measurement

{unit of measure}

e.g.

MM

The type of measurement given for the cell definitions e.g. MM.

(For a full list of available units see unit of measure)
