queryFilters Tag


Contents

The queryFilters tag holds a list of queryFilter tags.

 

A query Filter can be used to filter (i.e. hide information we are not interested in)

 

Example queryFilters XML

<queryFilters>
      <queryFilter name="majorComponentFilter" >
      appliesto bus type Part name * revision * where 'attribute[Component Name] == "${requestParamter:Component}" AND attribute[Component Type] ~~ "${requestParamter:ComponentType}"'
      </queryFilter>

</queryFilters>
