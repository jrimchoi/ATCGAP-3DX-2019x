ActionActivity Actions


Contents

The 3dxtools ActionActivity Actions provides the ability to to execute the independent action activity as a child action. If the action activity returns failure then the action will not pass and further child actions will not be executed.

ActionActivity actions will not block the current operation. But if any dependant check action returns failure then current operation will be blocked and child action’s validation message will be displayed to the user.

 

Configuration and Settings

To setup a 3dxtools ActionActivity Actions:

1. Setting Configuration ➣

•Add high level business logic in below format to any Action Activity object Settings attribute.

<actions>

 <actionActivity name="CheckLatestStatusOnVersion"/>

</actions>

 

2. Setting Details  ➣

Setting

Description

name

Name of the action activity object.

 

System will execute the actions defined in setting of given action activity object.

actions

child actions configuration

 

•For child actions, add any Action Activity you wish to run from this event.
