style Tag


Contents

The style tag defines preset text formatting options that can be used in other tags, specifiying the font, size, format and colour to be used.

 

style attributes:

Name

Value

Notes

name

{Suitable value}

The name of the style, so it can be referenced later.

font

{Suitable font name}

The font to be used in the style. The font must be installed on the server.

size

{Suitable numeric value}

The size of text used in the style.

format

{Any combination of bold,
italic,underline}

e.g.

bold,underline

The text formatting to be used in the style.  Can hold multiple values separated by a comma and no space.

default

true
false

Defines if the style is default or not. Must only be true for one style.

colour

{Suitable colour}

The colour of text in the style.

 

Example style XML

<style name="bold" font="Arial" size="9" format="bold"/>
<style name="boldUnderline" font="Arial" size="9" format="bold,underline"/>
<style name="normal" font="Arial" size="9" format="" default="true"/>
<style name="large" font="Arial" size="14" format="" />
<style name="H1" font="Arial" size="14" format="bold" />
<style name="H2" font="Arial" size="12" format="bold" />
<style name="small" font="Arial" size="5" format="" />
<style name="smallbold" font="Arial" size="5" format="bold" />
