leftHandValue Tag


Contents

The leftHandValue tag is the value that the rowIf will use to populate the left hand side of the if condition with, using the leftHandSource value to instruct it where to fetch the value from.

 

Example leftHandValue XML

This example will compare a value returned from an MQL command.

<leftHandSource>mql command</leftHandSource> 
<leftHandValue>eval expression 'count TRUE' on expand bus ${ID} from relationship "Major Component" type "Part" recurse to 1 filter majorComponentFilter</leftHandValue>
