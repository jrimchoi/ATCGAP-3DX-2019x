foreach Tag


Contents

A foreach tag holds a query that will populate the document with values for each value returned in the MQL query result. A foreach tag can have a nested foreach tag that can run a further query on the result, such as expand statements.

 

Attributes:

Name

Value

Notes

query

{suitable query}

The query to be run

objectIdPosition

{numerical value}

The position of the object id that is returned by the query

orderby

{numerical value}

The position of a returned value in the query that the query is ordered by

dumpDelimiter

{Suitable character}

e.g. a pipe character:

|

The character used to delimit the query results
