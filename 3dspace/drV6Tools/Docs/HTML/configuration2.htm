Configuration


Contents

 This feature of 3dxTools will provide flexibility and framework for adding new routes based on addition of series of business object types.  DRTOOLS Initiate Route will provide provision to choose which route templates use during the creation of Route objects. The XML setting defined in drlSettings of DRLTools Route Setting object will define the operations to be performed on DRL DOCUMENTS during route creation.  These settings provide great flexibility and ENOVIA administrator overhead.

 

Note: 3dxtools Route Enhancement will be disabled by default. If you want to enable the functionality then follow steps mentioned in Enable Configuration

 

This feature has provided route configuration which can be done in ENOVIA web interface using administrator.
