Email Scanner Activity


Contents

The 3dxtools Email Scanner Activity is used to scan Outlook emails (*.msg files) as they are checked into ENOVIA, extract details like the message subject, body, sender and recipients, and map the details to the current ENOVIA object. It can also extract any file attachments contained within the email and check them into the business object along with the original email *.msg file.

Configuration and Settings

To setup a 3dxtools Email Scanner Activity:

1. Add a drV6tools Email Scanner Activity object ➣

•Select the Dependant Activities tab on either a top level 3dxtools Triggers action object or another 3dxtools Activities object.
•Click on Actions.
•Select Create DRLTools Email Scanner Activity...

oThis will open the DRLTools Email Scanner Activity create form.
•Set the following values:

Use Shadow Agent

TRUE

or

FALSE

TRUE (ticked) means the system will use the background system super user to carry out the email scanning activity. This may be needed if standard users do not have the required permissions to update a business object and check in file attachments.

FALSE (unticked) means the system will not use the background system super user to carry out the email scanning activity.
