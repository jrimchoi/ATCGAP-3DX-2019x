2. Create Type Action Trigger object

To Add the Create Event Trigger to the DRL DOCUMENTS Type in ENOVIA

•In ENOVIA Business find the ENOVIA DRL DOCUMENTS type.
•On the Triggers tab for DRL DOCUMENTS type find the ENOVIA Create trigger
•Edit the trigger.
•Set the following values:

Name

Value

Notes

Action

emxTriggerManager

This should already be set for you

Input

TypeAllDRToolsV6RunActionTrigger

Append to any existing value

TypeAllDRToolsV6RunActionTrigger_Deferred waits until the current transaction scope has completed before running the dependant 3dxtools Activities.

 

To create a DRLTools Type Action Trigger:

•Open DRL DOCUMENTS - Checkin Actions placeholder DRLTools Folder and select the Folder Content tab
•Click Actions
•Select the Create DRLTools Type Action Trigger

•Fill the following details in Create DRLTools Type Action Trigger form
•The fields are explained below.

Name

Value

Notes

Type

DRLTools Type Action Trigger

This should already be set for you

Name

Checkin_DRL DOCUMENTS

 

{Operation Name}_{Type Name}

{Operation Name} should be set to the trigger in ENOVIA you wish to run the activities from (for instance Create, Modify, Checkin, etc)

{Type Name} should be set to the ENOVIA object type the trigger is being run on.

It will execute the dependant activities for ENOVIA Create event.

Description

3dxtools DRL DOCUMENTS Check In

Any suitable value
•Click on Done.

DRLTools Type Action Trigger is created successfully.
