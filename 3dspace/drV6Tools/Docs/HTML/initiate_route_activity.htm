Initiate Route Activity


Contents

The 3dxtools Initiate Route Activity is a simple mechanism to initiate routes using pre-defined route templates on the currently selected object from a standard ENOVIA web command or operation trigger.

 

Configuration and Settings

To setup a 3dxtools Initiate Route Activity:

1. Add/Find the Route Template in ENOVIA ➣

Find (or if required create) the Route Template that the the Initiate Route Activity will initiate, and note down its name value. For details on how to find or create Route Templates please see the standard ENOVIA documentation.

2. Add a drV6Tools Initiate Route Activity object ➣

•Select the Dependant Activities tab on either a top level 3dxtools Triggers action object or another 3dxtools Activities object.
•Click Actions.
•Select Create DRLTools Initiate Route Activity...

oThis will open the DRLTools Initiate Route Activity create form.
•Set the following values:

Route Base Purpose

{A valid Route Base Purpose value}

By default the system will use the Route Base Purpose defined in the Route Template but it is possible to override this value for the Initiate Route Activity using this value.

Route Completion Action

{A valid Route Base Completion Action value}

By default the system will use the Route Completion Action defined in the Route Template but it is possible to override this value for the Initiate Route Activity using this value.
