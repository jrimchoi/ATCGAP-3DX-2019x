designrule’s 3DEXPERIENCE ENOVIA Productivity Toolset 3dxtools significantly reduces the implementation time of 3DEXPERIENCE ENOVIA PLM solutions. By delivering pre-built and configurable enhancements to standard 3DEXPERIENCE ENOVIA functionality, 3dxtools results in faster implementations and higher returns on PLM investments.

Gaps will always exist between what users require and what the OOTB (out-of-the-box) 3DEXPERIENCE ENOVIA functionality provides. To address these gaps, implementation teams need to evaluate the impact of changing the product configuration on upgradeability, interoperability and implementation time. These changes make implementations and upgrade longer and more expensive.

During its 50+ PLM implementations, Design Rule realized that most organizations require similar changes to OOTB PLM solutions like 3DEXPERIENCE ENOVIA so we developed this new Toolset architecture for customers who want an OOTB solution, but need it configured for their business. 3dxtools is derived from the successful DRTools developed for ENOVIA Smarteam, that is used in hundreds of companies worldwide and delivers OOTB++

3dxtools enables individual enhancements to be componentised, simplifying common changes while streamlining their implementation and upgrades. 3dxtools not only closes the gap between requested and OOTB 3DEXPERIENCE ENOVIA functionality, but also reduces the time and cost required to investigate the impact of each change, leading to faster and lower risk implementations, while typically eliminating the costs of customisation services.

designrule 3DEXPERIENCE ENOVIA toolkit (3dxtools) is a series of tools, utilities and applications designed to reduce the implementation time of ENOVIA implementations whilst increasing the end user experience.

The toolkit is split into three main installations:

•3dxtools are a series of highly configurable utilities that can be coupled together to perform complex actions typically required in an 3DEXPERIENCE ENOVIA implementation.
•3dxprocessor Job Server is a general multi-purpose server side job processing platform designed to handle any offline task required by an 3DEXPERIENCE ENOVIA implementation (e.g. PDF view file generation).
•3dxtools Utilities is a series application designed for the rapid configuration and management of ENOVIA implementations.
