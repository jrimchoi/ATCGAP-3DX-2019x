Contents

The 3dxtools Merge PDFs feature helps in merging all PDF files attached to documents and downloading in once click. This feature is used specially to download all the files for taking a printout. If user has to download each files individually, it will take considerable amount of more time and also there is a chance of missing few files when there are large number of documents in workspace.

Merge PDF feature of 3dxtools can also be used to prepare reports with all files in workspace put together in one file with table of content mentioning details of individual files. User can also stamp the pages of PDFs as required. For example, each files can be stamped with the corresponding documents current state and name of the user/owner of the document.

In this example, we are downloading the PDF files attached to the documents in workspace in a single PDF and also downloading with Table of Content. Also we are stamping the file with current state of the documents.

Following are the 3dxtools functionalities covered in this walkthrough:

1.Create DRLTools Folder object

The 3dxtools Folder is the place holder object for 3dxtools triggers. It is used to combine and display the configured activities for a standard ENOVIA operation in an expanded view.

 In this example, we will create a placeholder for ENOVIA Create operation.

2.Create Type Action Trigger object

The 3dxtools Type Action Trigger is used to connect an action event on a Type in ENOVIA.

In this example, we will add Create trigger in the Download command in workspace actions. This will combine all the PDF files available on the selected document object. If there are not PDF files available, Error message is displayed.

3.Create DRLTools Export Activity object

The 3dxtools Export Activity feature is used to export and publish the data in xml format and files from Enovia using trigger object. The system provides the mechanism where user can define what attributes, related objects and files to be exported. If it is set to download, then the system will combine all the files and adds to zip file for download.

In this example, we will be downloading multiple files in a zip file from the workspace containing multiple documents.

4.Create DRLTools Export Target Download object

This drV6Tool feature is used to define the target for metadata or files exported by the Export Activity. Using this, user can define variable directory location to export data.

 In this example, we export the data in the form of xml.

5.Create DRLTools PDF Merger Activity

This feature is used to merge the PDF files attached to multiple documents and download them in a single file.  This will help user in individually downloading the files from multiple documents and reduce the execution time.

 In this example, we will use this feature to download PDF files from multiple documents to single file to print.

6.Create DRLTools PDF Stamper Activity 

This feature of 3dxtools is used to add the watermark on the pages of PDF files downloaded from Workspace. In this example, we are adding the state of the document along with name, revision of the document. The stamping positioning is defined using the xml configuration on this object.

7.Create DRLTools PDF Report Activity 

The 3dxtools PDF Report Activity allows for a user to create a detailed report, showing all or some attributes on an object and any number of relationships those objects have. The report is built up from an XML configuration file and if necessary Copy Values Target Attribute objects.

The report generated is a PDF Document showing the objects that were requested in an MQL query defined in the xml configuration.

8.Create DRLTools Copy Values Target Attribute 

This feature is used to copy/update the values on the target attribute. It will fetch the value based on the conditions and paste on the target attribute.

 In this example, we are stamping the current time on the PDF report.

9.Create DRLTools Copy Values Element Date Value 

This feature is used to copy the current date to target object. This will fetch the system date and update on the target attribute.

 In this example, we use this feature to update the date on the report.
