3dxtools Triggers


Contents

3dxtools Triggers provide a way of connecting triggers in ENOVIA with the 3dxtools Activities administrators may wish to run.  There are two types of trigger:

 

•ENOVIA event triggers (executed by the ENOVIA core based on events on the business objects or relationships):
oType Action Trigger
oRelationship Action Trigger
oAttribute Action Trigger
oPolicy State Action Trigger

 

•ENOVIA user command triggers (executed when users select commands within the ENOVIA web user interface):
oCommand Action Trigger

 

ENOVIA Triggers are then connected with 3dxtools Trigger Action objects, which then allow you to define dependent 3dxtools Activities to run when they are executed.
