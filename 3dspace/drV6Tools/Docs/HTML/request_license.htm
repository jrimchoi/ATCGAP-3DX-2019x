Request License


Contents

This command is used to request for new license for 3dxtools and components. User has to enter the details in all mandatory fields to request for new license. For renewal, the request license page will be auto filled from the previous license.

From Home page, navigate to Designrule Tools Admin à 3dxtools License à Request License.  This will open the below page.

 

 Fields highlighted in red italic are mandatory fields.

SL No

Field Name

Details

1

3dxtools Build Version

This is the build number of the installed 3dxtools. This will be non-editable and auto populated from the 3dxtools installation.

2

License Type

This is the type of license. It can have the following values. Activated/Temp/Fixed version. Default value will be temp at fresh installation. This field is non-editable.

 

Temp: This will have 30 days validity and user must request for the new license.

 

Fixed version: This license is valid for the fixed version mentioned in the license. This license cannot be used for different version of 3dxtools.

 

Activated: This is the activated version of license which provided all features of licensed product till the expiry date.

3

Licensed Products

This provides the information of products for which the license is available.

Default values will be 3dxdesigner, 3dxcomponents, 3dxprocessor, 3dxtools at fresh installation.

This field is non-editable.

 

4

License Expiry Date

Expiry date of the license. This will be 30 days from the time of the day of fresh installation of 3dxtools. This field is non-editable.

5

Locations Count

Active locations available in the environment. Default value will be 1. If there are many locations available in system, it will auto calculate and populate the value.  This field is non-editable.

6

Active Users Count

Active users available in the environment. Auto calculated.  This field is non-editable.

7

Environment Name

Editable field where user should enter name of the environment.

8

Environment Type

Editable field where user should select the type of environment. It can have the following options to select from combo box: Production/Test/Validations/Development

9

Customer Name

Editable field where user should enter Name of the customer who is requesting for the license.

10

Customer Address

Editable field where user should enter Communication address of customer.

11

Customer Contact Name

Editable field where user should enter Name of point of contact

12

Customer Contact Email Address(es)

Editable field where user should enter Email address of point of contact. It can be multiple separated by comma.

13

VAR Name

Editable field where user should enter Name of VAR

14

VAR Contact Name

Editable field where user should enter Name of point of contact of VAR

15

VAR Contact Email Address(s)

Editable field where user should enter Email address of point of contact of VAR

16

Request Notes

Editable field where user may enter Notes for requesting the license. Non-mandatory field.

 

 Once all the details are entered in the above fields, click on Submit. This will generated the request XML as shown below.

 

This page has two commands. Copy to clipboard and Close. User should to copy (use copy to clipboard button to avoid missing any data of request XML) the request XML and email it to drlTools@designrule.co.uk
