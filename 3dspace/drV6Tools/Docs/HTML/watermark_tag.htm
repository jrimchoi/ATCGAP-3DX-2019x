watermark Tag


Contents

The watermark tag holds all the information to create a watermark or stamp on the PDF document. It is made up of a series of attributes and a defined value.

 

Attributes:

Name

Value

Notes

style

{Suitable style}

A style that has been previously defined.

horizontalPosition

{left, right or middle}

The position horizontally.

verticalPosition

{top, center or bottom}

The position vertically.

alignment

{left, right or center}

The alignment of the text within the stamp.

rotation

{Suitable number}

The rotation of the stamp in degrees.

height

{Suitable number}

The height of the stamp in pixels.

width

{Suitable number}

The height of the stamp in pixels.

underlay

true

or

false

true means the stamp will be underneath the document text.

false means the stamp will be on top of the document text.

x

{position in points}

These two attributes allow you to position the watermark using X and Y coordinates defined in points.

If using this option then the other vales (horizontalPosition, verticalPosition, alignment, rotation, height and width) will not be used.

y

{position in points}

 

The watermark tag can contain the following sub-tags:

Name

Value

Notes

text

Suitable text value

This will display the exact text value.

value

Attribute Name

The value of an attribute of the returned object.

mql-selectable

mql selectable

This will be a selectable value on the returned object.

 

Example watermark XML

<watermark style="large" horizontalPosition="middle" verticalPosition="center" alignment="center" rotation="45" height="200" width="800" underlay="false"> 
   <value>Title</value><text> - </text><mql-selectable>current</mql-selectable> <!-- this is the mql selectable value that gets the value of the attribute that you wished to be stamped -->
</watermark> 
<watermark style="H1" horizontalPosition="middle" verticalPosition="bottom" alignment="left" height="25" width="400" underlay="false"> 
   <text>Printed by: </text><value on="user">First Name</value><text> </text><value on="user">Last Name</value> 
</watermark> 
<watermark style="large" x="100" y="200" underlay="false"> 
   <text>Property Of...</text> <!-- A fixed text value -->
</watermark>
