Custom Processor


Contents

The 3dxprocessor Custom Processor provides a mechanism to executed additional custom processors developed outside the 3dxtools within the 3dxprocessor framework. To use this you must first create and compile a java class that inherits from com.designrule.drv6tools.jobserver.drJobServerCustomProcessor. This should be built into a JAR file that should be placed into the .\Program Files\Designrule\3dxprocessor\drV6Tools.JobServer.Service\lib directory before you create the 3dxprocessor instance. You then just have to create a Custom Processor object in the database and set the attribute Java Class Name to the full class name of the custom processor java class. A series of optional arguments have been created on the business object to allow custom settings to be stored for the processor.
