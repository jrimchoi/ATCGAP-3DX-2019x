fixedText Tag


Contents

The fixedText tag contains a series of sub tags whose values will be written from the start of the page for a known length. Fixed text tags are populated using value tags.

To display a textual value in the report, a value tag is needed. This value can be retrieved from a variety of sources, such as; mql-selectables, attributes, or even a plain text value. Each value can be formatted using a format tag that holds the value tag inside it. The following tables and examples detail how to construct these.

Examples:

The following details how to populate a cell in a table with text values.

 

<cell backgroundcolour="rgb(220,220,220)" columnspan="2">
     <format type="decimal" formatString="#.##">

         <mql-selectable>attribute[Part Length].unitvalue[m]</mql-selectable>

 </format>
           <text> m</text>

</cell>

 

<cell rowspan="1" columnspan="6" style="boldUnderline">
      <text>Physical Characteristics</text>
</cell>

 

<cell rowspan="1" columnspan="2" evenbackgroundcolour="rgb(220,220,220)">
    <value>Structured Data Sheet Boolean</value>
    <value>Structured Data Sheet Text2</value>
    <value>Structured Data Sheet Number2</value>
</cell>

 

This next example shows how to define flow text:

 

<flowtext>
 <value>Structured Data Sheet Boolean</value>
 <value>Structured Data Sheet Text2</value>
 <value>Structured Data Sheet Number2</value>
</flowtext>

 

This final example shows how to use formats inside a header

<headertext style="bold" horizontalPosition="right" verticalPosition="bottom" alignment="left" height="50" width="150">
 <text>Date: </text>

 <format formatString="dd-MMM-yyyy" type="date">

         <value>Originated</value>

 </format>
</headertext>
