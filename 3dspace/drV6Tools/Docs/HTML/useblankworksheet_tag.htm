useBlankWorksheet Tag


Contents

The useBlankWorksheet tag tells the report whether it is going to get written to a blank sheet in the document or a predefined one. Its value is either true or false. If the value is set to true, the worksheetName tag is not required to be set.

 

Example worksheetName XML

<useBlankWorksheet>False</useBlankWorksheet>
